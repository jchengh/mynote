Vim 的核心哲学是**“像交流一样编辑”**，其命令通常遵循 `[数字] + [谓词/操作] + [名词/范围]` 的逻辑。

---

## 一、 模式切换 (Mode Switching)

Vim 的力量源于不同的模式，你需要频繁在它们之间切换。

|命令|英文全称|描述/示例|
|---|---|---|
|`i`|**I**nsert|在光标前插入文本|
|`a`|**A**ppend|在光标后追加文本|
|`o`|**O**pen line|在当前行下方新开一行插入|
|`Esc`|**Esc**ape|返回普通模式 (Normal Mode)|
|`:`|**Colon**|进入命令行模式 (Command-line Mode)|
|`v`|**V**isual|进入可视化模式（按字符选择）|

## 二、 光标移动 (Motion)

在普通模式下，移动是最高频的操作。

### 1. 基础位移 (Basic)

- **`h` / `j` / `k` / `l`**: 左 / 下 / 上 / 右。
    

### 2. 单词级移动 (Word-wise)

- **`w`** (**W**ord): 跳转到下一个单词的开头。
    
- **`b`** (**B**ackward): 跳转到当前或上一个单词的开头。
    
- **`e`** (**E**nd): 跳转到当前或下一个单词的结尾。
    

### 3. 行内与跨行移动 (Line & Page)

- **`0`**: 跳转到行首。
    
- **`$`**: 跳转到行尾。
    
- **`gg`**: 跳转到文件开头。
    
- **`G`**: 跳转到文件末尾。
    
- **`5G`**: 跳转到第 5 行。

## 三、 编辑操作 (Editing)

这部分是实战演示中的重点，遵循 `操作 + 范围`。

|命令|英文全称|示例|
|---|---|---|
|`x`|(Delete) **x**|删除光标处的单个字符|
|`dw`|**D**elete **W**ord|删除从光标到单词末尾的内容|
|`dd`|**D**elete **D**ouble|删除整行|
|`yy`|**Y**ank **Y**ank|复制整行|
|`p`|**P**ut/Paste|在光标后粘贴已复制/删除的内容|
|`u`|**U**ndo|撤销上一步操作|
|`Ctrl + r`|**R**edo|重做（反向撤销）|
|`r`|**R**eplace|替换光标下的字符（例如 `ra` 将当前字符换成 a）|

## 四、 文本对象操作 (Text Objects)

这是 Vim 的“黑魔法”，适合修改括号、引号内的内容。格式：`[操作] + [i/a] + [对象]`。

- **`ci"`** (**C**hange **I**nside **"**): 修改双引号内部的内容。
    
    - _示例_：`print("Hello World")`，光标在引号内时输入 `ci"`，会清空字符串并进入插入模式。
        
- **`di(`** (**D**elete **I**nside **(**): 删除括号内的所有内容。
    
- **`ca{`** (**C**hange **A**round **{**): 修改大括号及大括号本身。
    

---

## 五、 命令行模式 (Command-line Mode)

通过 `:` 触发，处理文件保存和全局设置。

- **`:w`** (**W**rite): 保存文件。
    
- **`:q`** (**Q**uit): 退出（若未保存会报错）。
    
- **`:wq`**: 保存并退出。
    
- **`:q!`**: 强制退出，不保存。
    
- **`:%s/old/new/g`**: 全局搜索并将 `old` **S**ubstitute（替换）为 `new`。

### 一、 插入模式下的 `Ctrl` 快捷键

这些命令能让你在输入文字时，减少按 `Esc` 的次数，保持打字流。

|**命令**|**英文描述**|**功能说明**|
|---|---|---|
|**`Ctrl + h`**|**B**ackspace|删除光标前的一个字符（等同于退格键）。|
|**`Ctrl + w`**|Delete **W**ord|删除光标前的一个单词。|
|**`Ctrl + u`**|Delete **U**p to line start|删除当前行光标前的内容（整行清除最常用）。|
|**`Ctrl + o`**|**O**ne command|**神技**：临时切换到普通模式执行**一个**命令，然后自动回到插入模式。|
|**`Ctrl + n`**|**N**ext match|自动补全：向下搜索文档中出现过的单词进行补全。|
|**`Ctrl + p`**|**P**revious match|自动补全：向上搜索单词进行补全。|

> **操作示例**：
> 
> 如果你在输入代码时想临时移动一下位置，按 `Ctrl + o`，然后输入 `zz`（将当前行置于屏幕中央），你会发现你依然停留在插入模式，可以继续打字。

### 二、 行号与鼠标的设置

这些设置属于环境配置，在 Vim 内部通过输入 `:` 进入命令行模式设置。

#### 1. 行号设置 (Line Numbers)

- **显示行号**：`:set number` (简写 `:set nu`)
    
- **取消显示**：`:set nonumber` (简写 `:set nonu`)
    
- **相对行号**（Vim 高手必备）：`:set relativenumber` (简写 `:set rnu`)
    
    - _提示：这会显示当前行与其他行的距离，方便配合 `5j`、`10k` 等命令快速跳转。_
        

#### 2. 鼠标设置 (Mouse Support)

虽然使用 Vim 的终极目标是抛弃鼠标，但在学习初期开启鼠标点击和滚动会很有帮助：

- **开启鼠标**：`:set mouse=a`
    
    - 参数 `a` 代表 **A**ll 模式（包括普通、插入、可视化模式）。开启后你可以用鼠标点击定位、滚动滚轮或拖动选择。
        
- **关闭鼠标**：`:set mouse=`

### 三、 永久生效：修改配置文件

如果你不想每次打开 Vim 都手动输入这些命令，你需要修改 WSL 中的 `.vimrc` 文件：

1. 在 WSL 终端输入：`vim ~/.vimrc`
    
2. 进入插入模式，将以下内容粘贴进去：
    
    Vim Script
    
    ```
    " 显示绝对行号
    set number
    " 同时开启相对行号（混合模式，当前行显示绝对行号，其他显示相对）
    set relativenumber
    " 开启鼠标支持
    set mouse=a
    " 解决从网页复制代码到 Vim 格式乱掉的问题
    set paste
    ```
    
3. 保存退出 (`:wq`)。
