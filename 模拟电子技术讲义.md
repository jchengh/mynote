## 第1章 半导体基础知识

### 1.1 半导体的导电特性

#### 1.1.1 本征半导体

半导体是导电能力介于导体和绝缘体之间的物质。常用的半导体材料有硅(Si)和锗(Ge)。

**本征半导体**：完全纯净的、晶体结构完整的半导体。

在绝对零度时，本征半导体中的价电子被共价键牢牢束缚，不能自由移动，因此不导电。当温度升高或受到光照时，部分价电子获得足够能量挣脱共价键的束缚成为**自由电子**，同时在共价键中留下一个空位，称为**空穴**。

```
温度T=0K
Si → Si + e⁻ (自由电子)
     ↑  ↑
     |  └── 空穴

自由电子和空穴成对出现，称为"电子-空穴对"
```

**载流子**：运载电荷的粒子。
- 电子：带负电的载流子
- 空穴：带正电的载流子

本征半导体中，自由电子和空穴浓度相等，且随温度升高而指数增加：

$$n_i = p_i = A \cdot T^{3/2} \cdot e^{-E_g/(2kT)}$$

其中：
- $n_i$：自由电子浓度
- $p_i$：空穴浓度
- $A$：与半导体材料有关的常数
- $E_g$：禁带宽度（硅约1.12eV，锗约0.67eV）
- $k$：玻尔兹曼常数
- $T$：热力学温度

#### 1.1.2 杂质半导体

通过掺杂工艺，可以改变半导体的导电性能，形成**杂质半导体**。

**1. N型半导体（电子型半导体）**

在硅中掺入五价元素（如磷P、砷As），五价元素的原子有5个价电子，其中4个与硅形成共价键，多余的一个电子很容易成为自由电子。

```
施主杂质           多数载流子：电子
P → P⁺ + e⁻       少数载流子：空穴
```

特点：
- 多数载流子(majority carrier)是电子
- 少数载流子(minority carrier)是空穴
- 电子浓度 ≈ 掺杂浓度 $N_D$

**2. P型半导体（空穴型半导体）**

在硅中掺入三价元素（如硼B、镓Ga），三价元素的原子有3个价电子，与硅形成共价键时缺少一个电子，形成一个空穴。

```
受主杂质           多数载流子：空穴
B → B⁻ + h⁺       少数载流子：电子
```

特点：
- 多数载流子是空穴
- 少数载流子是电子
- 空穴浓度 ≈ 掺杂浓度 $N_A$

**重要结论**：
- N型半导体中，电子是多子，空穴是少子
- P型半导体中，空穴是多子，电子是少子
- 杂质半导体的导电能力远高于本征半导体

### 1.2 半导体的导电机制

#### 1.2.1 载流子的漂移运动

在电场作用下，载流子会发生定向移动，称为**漂移运动**。

**漂移电流密度**：
$$J = q(n\mu_n + p\mu_p)E$$

其中：
- $q$：电子电荷量（1.6×10⁻¹⁹ C）
- $\mu_n$：电子迁移率（硅中约1350 cm²/(V·s)）
- $\mu_p$：空穴迁移率（硅中约500 cm²/(V·s)）
- $E$：电场强度

**电导率**：
$$\sigma = q(n\mu_n + p\mu_p)$$

#### 1.2.2 载流子的扩散运动

当载流子浓度分布不均匀时，载流子会从高浓度区向低浓度区扩散，形成**扩散电流**。

**扩散电流密度**：
$$J_n = qD_n \frac{dn}{dx}$$
$$J_p = -qD_p \frac{dp}{dx}$$

其中 $D_n$、$D_p$ 为扩散系数。

### 1.3 温度对半导体的影响

温度对半导体导电性能的影响极为显著：

| 参数 | 温度升高时 |
|------|-----------|
| 本征激发 | 增加 |
| 少子浓度 | 指数增加 |
| 电阻率 | 减小 |
| 热激发 | 增加 |

温度每升高10°C，本征载流子浓度约增加一倍。这就是半导体器件热稳定性差的根本原因。

## 第2章 PN结与半导体二极管

### 2.1 PN结的形成

#### 2.1.1 载流子的浓度差与扩散

当P型半导体和N型半导体结合在一起时，由于两侧多子浓度差异：

- P区的空穴向N区扩散
- N区的电子向P区扩散

扩散结果：
```
P区                    N区
空穴(-) ←──→ 电子(+)

扩散前：
P区：[  空穴浓度高  ] [  电子浓度低  ]N区

扩散后：形成空间电荷区（耗尽层）
P区：[ - - - - - ]|[+ + + + + ]N区
       负离子区      正离子区
```

#### 2.1.2 空间电荷区与内电场

扩散运动在交界面附近形成一个区域，其中载流子被复合耗尽，只剩下不能移动的杂质离子，这个区域称为**空间电荷区**或**耗尽层**。

空间电荷区产生内电场，方向由N区指向P区。内电场的作用：
1. 阻碍多子继续扩散
2. 促进少子漂移

#### 2.1.3 动态平衡

当扩散作用与漂移作用达到动态平衡时，PN结达到稳定状态。此时：
- 扩散电流 = 漂移电流（方向相反）
- 净电流 = 0
- 空间电荷区宽度一定（约0.5μm）

### 2.2 PN结的单向导电性

#### 2.2.1 正向偏置

当P区接高电位、N区接低电位时，称为**正向偏置**（Forward Bias）。

```
电路连接：
  +  ──→── [P]──[N]───  -
  R      ↓           ↓
  │      ↓           ↓
  └──────┴───────────┘
```

**特点**：
- 外电场与内电场方向相反
- 外电场削弱内电场
- 空间电荷区变窄
- 多子扩散加剧，形成**正向电流**
- 正向电阻小

**正向电流方程（肖克利方程）**：
$$I_F = I_S \left( e^{\frac{V_F}{V_T}} - 1 \right)$$

其中：
- $I_S$：反向饱和电流（约10⁻¹⁵~10⁻¹²A）
- $V_T = \frac{kT}{q}$：热电压（约26mV，室温下）
- $V_F$：正向电压（约0.6~0.7V，硅管）

#### 2.2.2 反向偏置

当P区接低电位、N区接高电位时，称为**反向偏置**（Reverse Bias）。

```
电路连接：
  -  ──→── [N]──[P]───  +
  R      ↓           ↓
  │      ↓           ↓
  └──────┴───────────┘
```

**特点**：
- 外电场与内电场方向相同
- 外电场增强内电场
- 空间电荷区变宽
- 扩散几乎停止，少子漂移形成**反向电流**
- 反向电阻大

**反向电流**：
- 由少子漂移形成
- 数值很小（μA级）
- 受温度影响大，温度每升高10°C，反向电流约增加一倍

### 2.3 二极管的伏安特性

#### 2.3.1 伏安特性曲线

```
        I (mA)
        ↑
    0.7 ┤  ●
        │ ●
    0.6 ┤●
        │                      正向特性
    0.3 ┤●
        │●
        └─────────────┬──────────────→ V (V)
                       │
        0              │              0.7
        │              │
   -0.1 ┤              │  反向击穿区
        │              │
   -0.2 ┤ ─ ─ ─ ─ ─ ─ ┘
        │   击穿电压
  -10μA ┤
        │              |
        │              |
        └──────────────┴─────
              -10V
```

#### 2.3.2 正向特性

- 当 $V < V_{on}$（门坎电压，约0.5V）时，电流很小
- 当 $V \approx 0.6\sim0.7V$（硅管）时，电流明显增长
- 锗管的门坎电压约为0.1~0.2V

#### 2.3.3 反向特性

- 反向电压在一定范围内，反向电流基本不变（饱和）
- 当 $|V_R| > |V_{BR}|$ 时，发生**击穿**

#### 2.3.4 击穿类型

| 击穿类型 | 原理 | 特点 |
|---------|------|------|
| 雪崩击穿 | 碰撞电离 | 温度升高，击穿电压升高 |
| 齐纳击穿 | 场致发射 | 温度升高，击穿电压降低 |

**雪崩击穿**：掺杂浓度低，外电场使载流子加速，碰撞电离，产生连锁反应。

**齐纳击穿**：掺杂浓度高，空间电荷区窄，强电场直接破坏共价键。

### 2.4 二极管的电容效应

#### 2.4.1 势垒电容 $C_T$

空间电荷区相当于一个电容，其容量随外加电压变化：

$$C_T = \frac{\varepsilon S}{W}$$

其中 $W$ 为空间电荷区宽度，随反向电压增大而增大。

#### 2.4.2 扩散电容 $C_D$

扩散到对方区域的少子积累形成的电容，正向偏置时起主要作用。

**总电容**：
$$C_j = C_T + C_D$$

变容二极管利用势垒电容随电压变化的特性，可作为压控电容使用。

### 2.5 二极管的主要参数

| 参数 | 符号 | 含义 |
|------|------|------|
| 最大整流电流 | $I_F$ | 长期允许通过的最大正向平均电流 |
| 最高反向工作电压 | $V_R$ | 允许承受的最大反向电压（通常为击穿电压的1/2~2/3） |
| 反向电流 | $I_R$ | 在最高反向工作电压下的反向电流 |
| 最高工作频率 | $f_M$ | 超过此频率，二极管失去单向导电性 |
| 正向压降 | $V_F$ | 正向电流为额定值时的正向电压降 |
| 动态电阻 | $r_d$ | 工作点处交流电阻，$r_d = \frac{V_T}{I_D}$ |

## 第3章 二极管应用电路

### 3.1 二极管电路的分析方法

#### 3.1.1 理想二极管模型

- 正偏时：电压为0，相当于短路
- 反偏时：电流为0，相当于开路

```
电路符号：
  A ──▶┃K ── B
    阳极  阴极

等效电路：
正偏：短路  A ────── B
反偏：开路  A ──┬──── B
              │ X │
              └───┘
```

#### 3.1.2 恒压降模型

考虑二极管的正向压降：

$$V_D = \begin{cases} V_D(on) & \text{正偏} \\ 0 & \text{理想} \end{cases}$$

硅管 $V_D(on) \approx 0.7V$，锗管 $V_D(on) \approx 0.3V$

#### 3.1.3 折线模型

$$I_D = \begin{cases} \frac{V_D - V_D(on)}{r_D} & V_D > V_D(on) \\ 0 & V_D \leq V_D(on) \end{cases}$$

其中 $r_D$ 是线性区段的等效电阻。

### 3.2 整流电路

#### 3.2.1 半波整流

```
      T
AC ○──┤├──○──┬──○── R_L ──○
          │  │
          │  └──▶┃──┤
          │       │
          └───────┘

输出波形：
      V_o
       ↑
       │    ___     
       │   /   \   
       │  /     \  
       └────────────→ t
       0        π    2π
```

**输出电压平均值**：
$$V_{o(AV)} = \frac{V_{2m}}{\pi} = \frac{\sqrt{2}V_2}{\pi} \approx 0.45V_2$$

**输出电流平均值**：
$$I_{o(AV)} = \frac{V_{o(AV)}}{R_L}$$

**特点**：
- 电路简单
- 输出脉动大
- 变压器利用率低（约40%）

#### 3.2.2 全波整流

```
      T₁     D₁
AC ○──┤├──○──▶┃──┬──○── R_L ──○
          │    │  │
          │    │  └──▶┃──┤
          │    │     │ 
          └──○─┤├──○┘
          T₂     D₂

输出波形：
      V_o
       ↑
       │ /\    /\    
       │/  \  /  \   
       └────/\────→ t
       0    π    2π
```

**输出电压平均值**：
$$V_{o(AV)} = \frac{2V_{2m}}{2\pi} = \frac{2\sqrt{2}V_2}{\pi} \approx 0.9V_2$$

**特点**：
- 变压器需中心抽头
- 输出脉动频率是输入的2倍
- 变压器利用率较高

#### 3.2.3 桥式整流（最常用）

```
      D₁       D₂
AC ○──▶┃──┬───○───┃◀──○
        │  │   │   │
        │  └───┴───┘
        │      R_L
        └───┬───┬───┘
            │   │
      D₄ ◀──┘   └──▶ D₃
```

**特点**：
- 变压器无需中心抽头
- 变压器利用率最高
- 使用4个二极管
- 电流连续

### 3.3 滤波电路

#### 3.3.1 电容滤波

```
整流   D₁
AC ○──▶┃──┬───○───┬──○── R_L ──○
        │  │   │   │
        └──┴───┴───┘
            C
```

**工作原理**：
- 利用电容的充放电特性平滑输出电压
- 电容两端电压不能突变
- 当整流输出电压高于电容电压时充电，低于时放电

**特点**：
- 输出电压高（可达 $1.2V_2$）
- 轻负载时效果好
- 负载加重时输出电压下降

**电容选择**：
$$R_L C \geq (3\sim5)T$$

其中 $T$ 为交流电源周期。

#### 3.3.2 电感滤波

适用于大电流负载，可减小输出纹波。

#### 3.3.3 π型滤波

电容-电感-电容（C-L-C）组合，滤波效果最好。

### 3.4 限幅电路（钳位电路）

#### 3.4.1 二极管限幅器

```
     R
Vi ○──┬───○── Vo
      │   │
      └──▶┃──┘
          │
          ──
```

**串联限幅**：二极管与输出串联

**并联限幅**：二极管与输出并联

**双向限幅**：两个二极管反并联，限制输出在 ±V_D 范围内。

### 3.5 二极管保护电路

二极管可用于保护感性负载在断电时产生的反向电动势：

```
  线圈 L  ──┬── Vo
            │
            └──▶┃──┘
```

当线圈断电时，二极管提供电流通路，将反向电压钳制在 -0.7V 左右。

## 第4章 晶体三极管(BJT)

### 4.1 BJT的结构与分类

#### 4.1.1 基本结构

BJT由两个PN结、三个区域组成：

| 类型 | 结构 | 符号 |
|------|------|------|
| NPN | P区夹在两个N区之间 | ```N → P → N``` |
| PNP | N区夹在两个P区之间 | ```P → N → P``` |

三个电极：
- **发射极(Emitter)**：发射载流子，掺杂浓度高
- **基极(Base)**：控制载流子传输，基区薄
- **集电极(Collector)**：收集载流子，面积大

```
NPN型：
       C
       │
     ┌─┴─┐
     │ N │  集电区
     └─┬─┘
       │P  基区(薄)
     ┌─┴─┐
     │ N │  发射区
     └─┬─┘
       │
       E
```

#### 4.1.2 电流放大原理

以NPN管为例：

**外部条件**：
- 发射结正偏（$V_{BE} \approx 0.7V$）
- 集电结反偏（$V_{BC} < 0$）

**载流子传输过程**：
1. **发射区注人电子**：发射结正偏，电子从发射区注入基区
2. **基区复合**：注人基区的电子与基区空穴复合，形成基极电流 $I_B$
3. **集电区收集电子**：集电结反偏，漂移到集电结的电子被集电极收集

```
                发射结正偏              集电结反偏
发射区 → [N]━━━▶━━━[P]━━━▶━━━[N] ← 集电区
 e⁻→→→    扩散  →→   复合  →→  漂移 →→→ c⁻

I_E = I_B + I_C
I_C = β·I_B
I_E = (1+β)·I_B
```

### 4.2 BJT的电流分配关系

#### 4.2.1 直流电流放大系数

$$\bar{\beta} = \frac{I_C}{I_B} \quad \text{（共射极直流电流放大系数）}$$

$$\bar{\alpha} = \frac{I_C}{I_E} \quad \text{（共基极直流电流放大系数）}$$

**关系**：$\bar{\alpha} = \frac{\bar{\beta}}{1+\bar{\beta}}$

#### 4.2.2 交流电流放大系数

$$\beta = \frac{\Delta i_C}{\Delta i_B} \bigg|_{V_{CE}=const}$$

$$\alpha = \frac{\Delta i_C}{\Delta i_E} \bigg|_{V_{CB}=const}$$

### 4.3 BJT的特性曲线

#### 4.3.1 输入特性曲线

$$I_B = f(V_{BE}) \big|_{V_{CE}=const}$$

```
        I_B (μA)
        ↑
    20 ┤      V_CE = 5V
        │     /
    15 ┤    /  V_CE = 2V
        │   /
    10 ┤  /
        │ /
     5 ┤/
        └──────────────→ V_BE (V)
        0.5   0.6   0.7  0.8
```

**特点**：
- 与PN结正向特性类似
- $V_{CE}$ 增加，曲线右移（基区宽度调制效应）
- $V_{CE} \geq 1V$ 后，曲线基本重合

#### 4.3.2 输出特性曲线

$$I_C = f(V_{CE}) \big|_{I_B=const}$$

```
        I_C (mA)
        ↑
    60 ┤  I_B=60μA  ─────────
    50 ┤  I_B=50μA  ─────────
    40 ┤  I_B=40μA  ─────────
    30 ┤  I_B=30μA  ─────────
    20 ┤  I_B=20μA  ─────────
    10 ┤  I_B=10μA  ─────────
     0 ┤──────────────────────→ V_CE (V)
        0         10        20
        ↑         ↑         ↑
     饱和区   放大区     击穿区
```

**三个工作区域**：

| 区域 | 条件 | 特点 |
|------|------|------|
| 放大区 | 发射结正偏，集电结反偏 | $I_C = \beta I_B$，具有电流放大作用 |
| 饱和区 | 两个结都正偏 | $I_C < \beta I_B$，$V_{CE} \approx 0.3V$ |
| 截止区 | 两个结都反偏 | $I_B \approx 0$，$I_C \approx 0$ |

### 4.4 BJT的主要参数

| 参数 | 符号 | 典型值/说明 |
|------|------|-------------|
| 直流放大系数 | $\bar{\beta}$ | 20~200 |
| 交流放大系数 | $\beta$ | ≈ $\bar{\beta}$ |
| 反向电流 | $I_{CBO}$ | μA级 |
| 穿透电流 | $I_{CEO}$ | ≈ $(1+\beta)I_{CBO}$ |
| 集电极最大电流 | $I_{CM}$ | 超过时 $\beta$ 下降 |
| 集电极最大功耗 | $P_{CM}$ | $P_C = V_{CE} \cdot I_C$ |
| 反向击穿电压 | $V_{(BR)CEO}$ | 基极开路时C-E间耐压 |

### 4.5 温度对BJT的影响

1. **$V_{BE}$ 温度系数**：约 -2mV/°C（温度升高，$V_{BE}$ 减小）
2. **$\beta$ 温度系数**：温度升高，$\beta$ 增大（约+1%/°C）
3. **$I_{CBO}$ 温度系数**：温度每升高10°C，约增加一倍

## 第5章 基本放大电路

### 5.1 放大电路的组成

#### 5.1.1 放大电路的基本框图

```
  vi ──→ ┌────────┐ ──→ vo
          │ 放大器  │
  VCC ──→ │        │
          └────────┘
           ↑  ↑
         偏置  输入信号
```

**放大的本质**：用小的输入信号控制大的输出信号，实现功率放大。

**必要条件**：
- 有源器件工作在放大区
- 建立合适的静态工作点
- 输入信号能加到发射结上
- 输出信号能送到负载

#### 5.1.2 共射极放大电路

```
        VCC
         │
         R_C
         │
    C ───┼───○── Vo
         │   │
         │   └──▶┃ (C)
         │       │
        [B]──R_B─┤ (B)
         │       │    NPN
        [E]──────┘ (E)
         │
         ── GND

Vi ──C₁──┬───○──○── Vo
              B   C
```

### 5.2 静态工作点

#### 5.2.1 静态工作点的定义

**静态工作点(Q点)**：无输入信号时，晶体管各极直流电压和电流。

$$Q(I_{BQ}, I_{CQ}, V_{CEQ})$$

#### 5.2.2 直流通路与交流通路

**直流通路**：确定静态工作点
- 电容开路
- 电感短路
- 信号源短路（保留内阻）

**交流通路**：分析动态参数
- 大电容短路
- 直流电源短路

#### 5.2.3 静态工作点计算

$$I_{BQ} = \frac{V_{CC} - V_{BE}}{R_B} \approx \frac{V_{CC}}{R_B}$$

$$I_{CQ} = \beta I_{BQ}$$

$$V_{CEQ} = V_{CC} - I_{CQ}R_C$$

### 5.3 动态分析

#### 5.3.1 放大电路的主要性能指标

| 指标 | 符号 | 定义 |
|------|------|------|
| 电压放大倍数 | $A_v$ | $A_v = \frac{v_o}{v_i}$ |
| 电流放大倍数 | $A_i$ | $A_i = \frac{i_o}{i_i}$ |
| 输入电阻 | $r_i$ | $r_i = \frac{v_i}{i_i}$ |
| 输出电阻 | $r_o$ | $r_o = \frac{v_o}{i_o}\big|_{R_L=\infty}$ |
| 通频带 | $BW$ | 上限频率与下限频率之差 |

#### 5.3.2 图解法

1. **直流负载线**：$V_{CE} = V_{CC} - I_CR_C$
2. **静态工作点**：直流负载线与 $I_B = I_{BQ}$ 曲线的交点
3. **交流负载线**：通过Q点，斜率为 $-1/R_L'$ 的直线

```
        I_C
         ↑
    Q    ● 交流负载线
     ↗
    /  ↗ 
   /  / 
  /  /
 /  /    直流负载线
/  /
───────→ V_CE
  Q
```

#### 5.3.3 失真分析

| 失真类型 | 原因 | 现象 |
|---------|------|------|
| 饱和失真 | Q点过高，$i_C$ 正半周进入饱和区 | 输出波形底部被削平 |
| 截止失真 | Q点过低，$i_C$ 负半周进入截止区 | 输出波形顶部被削平 |
| 双向失真 | 输入信号过大 | 输出波形上下同时被削 |

### 5.4 静态工作点的稳定

#### 5.4.1 温度对工作点的影响

温度升高时：
- $\beta$ 增大
- $I_{CBO}$ 增大
- $V_{BE}$ 减小

导致 $I_C$ 增大，工作点上移，可能进入饱和区。

#### 5.4.2 分压式电流负反馈偏置电路

```
        VCC
         │
         R_B1
         │
    C ───┼───○── Vo
         │   │
         │   └──▶┃
         │       │
        [B]──R_E─┤
         │   │   │
         │   └───┤
        R_B2    C_E
         │       │
         ────────┘
```

**稳定原理**：
- $I_1 \gg I_B$，则 $V_{B} \approx \frac{R_{B2}}{R_{B1}+R_{B2}}V_{CC}$（固定）
- $V_E = V_B - V_{BE}$ 固定
- $I_E = V_E/R_E$ 固定
- $I_C \approx I_E$ 固定

**静态分析**：
$$V_B = \frac{R_{B2}}{R_{B1}+R_{B2}}V_{CC}$$
$$V_E = V_B - V_{BE}$$
$$I_E = \frac{V_E}{R_E}$$
$$I_{CQ} \approx I_E$$
$$V_{CEQ} = V_{CC} - I_{CQ}(R_C+R_E)$$

## 第6章 放大电路的分析方法

### 6.1 等效电路法

#### 6.1.1 BJT的h参数等效电路

从输入端和输出端特性出发，建立双口网络模型：

**h参数定义**：
$$h_{ie} = \left. \frac{\partial v_{BE}}{\partial i_B} \right|_{V_{CE}}$$（输入电阻）
$$h_{re} = \left. \frac{\partial v_{BE}}{\partial v_{CE}} \right|_{I_B}$$（反向电压传输系数）
$$h_{fe} = \left. \frac{\partial i_C}{\partial i_B} \right|_{V_{CE}}$$（电流放大系数）
$$h_{oe} = \left. \frac{\partial i_C}{\partial v_{CE}} \right|_{I_B}$$（输出电导）

**简化h参数等效电路**：

```
         b
          │
          ├─────── h_ie ──────┐
          │                    │
         h_re·v_ce           i_c
          │                    │
          └────────┬───────────┤
                   │           │
                  c◄──────────┘
                  /
                 /
                1/h_oe
               /
              ○ e
```

#### 6.1.2 简化的等效电路

在低频小信号条件下，忽略 $h_{re}$ 和 $h_{oe}$：

```
         b
          │
     v_be │ h_ie
          ├───────
          │       │
         r_be   βi_b
          │       │
          └───────┴───── c
```

其中：
- $r_{be} = r_{bb'} + (1+\beta)\frac{V_T}{I_E}$
- $r_{bb'}$：基区体电阻（约200~300Ω）
- $V_T$：热电压（约26mV）

**典型值**：$r_{be} \approx 300 + (1+\beta)\frac{26mV}{I_E(mA)}$ (Ω)

### 6.2 共射放大电路分析

#### 6.2.1 电压放大倍数

$$A_v = \frac{v_o}{v_i} = \frac{-\beta i_b R_L'}{i_b r_{be}} = -\frac{\beta R_L'}{r_{be}}$$

其中 $R_L' = R_C \parallel R_L$

**负号意义**：输出电压与输入电压相位相反（反相）

#### 6.2.2 输入电阻

$$r_i = R_B \parallel r_{be}$$

通常 $R_B \gg r_{be}$，所以 $r_i \approx r_{be}$

#### 6.2.3 输出电阻

$$r_o = R_C$$

### 6.3 直流通路与交流通路分析

#### 6.3.1 直流通路分析

将耦合电容开路，得到直流通路：

$$I_{BQ} = \frac{V_{CC} - V_{BE}}{R_B}$$
$$I_{CQ} = \beta I_{BQ}$$
$$V_{CEQ} = V_{CC} - I_{CQ}R_C$$

#### 6.3.2 交流通路分析

将耦合电容和旁路电容短路，直流电源接地：

- 输入回路：$v_i$ → $C_1$ → 基极 → 发射极 → $R_E$（旁路电容短路时）→ 地
- 输出回路：集电极 → $C_2$ → $R_C$ → 负载

         │
         R_B
         │
        [B]──┤ (B)
         │   │
         └───┼───○── Vi
             │   │
            [E]─R_E─┤ (E)

                 │

#### 7.2.2 性能特点

#### 7.2.3 应用

## 第8章 场效应管(FET)

### 8.1 结型场效应管(JFET)

#### 8.1.1 结构

```
         G(栅极)
          │
      ┌───┴───┐
      │  P区   │    N沟道JFET
      │(P区)   │
    ┌─┴────────┴─┐
    │   N区      │
    │  源极S     │
    │   N沟道    │
    └────────────┘
        D(漏极)
```

#### 8.1.2 工作原理

- $V_{GS}$ 控制沟道宽度
- $V_{DS}$ 产生漏极电流
- 当 $V_{GS}$ 足够负时，沟道夹断，$I_D = 0$

**转移特性**：
$$I_D = I_{DSS}\left(1 - \frac{V_{GS}}{V_P}\right)^2 \quad (V_P < V_{GS} < 0)$$

其中 $V_P$ 为夹断电压，$I_{DSS}$ 为饱和漏极电流。

### 8.2 绝缘栅场效应管(MOSFET)

#### 8.2.1 N沟道增强型MOSFET

```
         G(栅极)
          │
      ╔═══╧═══╗
      ║ SiO₂  ║
      ╠═══╤═══╣
      │ P区    │
    ┌─┴───────┴─┐
    │   N区     │
    │  源极S    │
    └───────────┘
        D(漏极)
```

**特点**：
- 栅极与沟道绝缘，输入电阻极高（>10¹⁴Ω）
- $V_{GS} = 0$ 时，$I_D = 0$
- 需要 $V_{GS} > V_{TH}$（阈值电压）才能形成沟道

**转移特性**：
$$I_D = K_n(V_{GS} - V_{TH})^2$$

其中 $K_n = \frac{\mu_n C_{ox}W}{2L}$ 为导电因子。

#### 8.2.2 N沟道耗尽型MOSFET

- $V_{GS} = 0$ 时已有沟道，$I_D = I_{DSS}$
- $V_{GS}$ 可正可负
- 负 $V_{GS}$ 减小沟道，正 $V_{GS}$ 增大沟道

### 8.3 FET的主要参数

| 参数 | 符号 | 含义 |
|------|------|------|
| 饱和漏极电流 | $I_{DSS}$ | $V_{GS}=0$ 时的漏极电流 |
| 夹断电压 | $V_P$ | JFET沟道夹断时的栅压 |
| 阈值电压 | $V_{TH}$ | MOS管开始导通的栅压 |
| 跨导 | $g_m$ | $g_m = \frac{\partial I_D}{\partial V_{GS}}$ |
| 漏极电阻 | $r_{DS}$ | 输出电阻 |

### 8.4 FET与BJT的比较

| 特性 | FET | BJT |
|------|-----|-----|
| 控制方式 | 电压控制 | 电流控制 |
| 输入电阻 | 极高 | 较低 |
| 噪声 | 较小 | 较大 |
| 温度稳定性 | 好 | 较差 |
| 功耗 | 低 | 较高 |
| 集成度 | 高 | 较低 |

## 第9章 功率放大电路

### 9.1 功率放大电路的特点

#### 9.1.1 主要问题

1. **输出功率大**：尽可能输出大的电压和电流
2. **效率高**：减少能量损耗
3. **非线性失真小**：输出波形不失真
4. **散热问题**：功率管的散热与保护

#### 9.1.2 功率指标

**输出功率**：
$$P_o = I_o \cdot V_o = \frac{1}{2}I_{om} \cdot V_{om}$$

**效率**：
$$\eta = \frac{P_o}{P_V} \times 100\%$$

其中 $P_V$ 为电源提供的直流功率。

### 9.2 功率放大电路的工作状态

| 类型 | 导通角 | 效率 | 失真 |
|------|--------|------|------|
| 甲类 | 360° | ≤50% | 小 |
| 乙类 | 180° | ≤78.5% | 交越失真 |
| 甲乙类 | 180°~360° | 50%~78.5% | 较小 |
| 丙类 | <180° | >78.5% | 大 |

### 9.3 乙类互补对称功率放大电路

#### 9.3.1 OCL电路（无输出电容）

```
        VCC
         │
      ┌──┴──┐
      │ NPN │
      └──┬──┘
         │T1
         ├────○─── Vo ─── R_L
         │
      ┌──┴──┐
      │ PNP │
      └──┬──┘
         │T2
         │
         ── GND
```

**工作原理**：
- $v_i > 0$：T1导通，T2截止，输出正半周
- $v_i < 0$：T2导通，T1截止，输出负半周

**最大输出功率**：
$$P_{om} = \frac{V_{CC}^2}{2R_L}$$

**效率**：
$$\eta = \frac{\pi}{4} \approx 78.5\%$$

#### 9.3.2 交越失真

乙类放大器在两管交替点附近存在失真（交越失真），原因是：

```
        Vo
         ↑
        /│\  交越
       / │ \ 失真
      /  │  \
──────────────→ t
```

**解决方法**：设置较小的静态偏置，使T1、T2工作在甲乙类状态。

### 9.4 甲乙类功率放大电路

#### 9.4.1 OTL电路（无变压器）

```
        VCC/2
         │
        R_L
         │
      ┌──┴──┐
      │ NPN │
      └──┬──┘
         ├────○─── Vo ──┤C│── R_L
         │
      ┌──┴──┐
      │ PNP │
      └──┬──┘
         │
         ── GND
```

**特点**：
- 单电源供电
- 输出电容耦合
- $C$ 起"负电源"作用

#### 9.4.2 OCL电路（改进型）

加入 $V_{BB}$ 偏置消除交越失真：

```
        VCC
         │
      ┌──┴──┐
      │ NPN │
      └──┬──┘
         ├────○─── Vo ─── R_L
         │
      ┌──┴──┐
      │ PNP │
      └──┬──┘
         │
         │T3
      R1│
      └──┴──○── V_BB
```

### 9.5 功放管的选用与保护

#### 9.5.1 安全工作区

- $P_{CM}$：最大允许功耗
- $I_{CM}$：最大允许电流
- $V_{(BR)CEO}$：反向击穿电压

#### 9.5.2 散热问题

- 功率管应安装散热器
- $T_j = P_C \cdot R_{th(j-c)} + T_c$
- 确保 $T_j < T_{jM}$（最大结温）

## 第10章 集成运算放大器

### 10.1 集成运放概述

#### 10.1.1 集成运放的特点

1. **高增益**：开环电压增益 $A_{vo}$ 可达 $10^5$ 以上
2. **高输入阻抗**：输入电阻 $r_i$ 可达 $10^6$Ω 以上
3. **低输出阻抗**：输出电阻 $r_o$ 约几百欧
4. **低漂移**：失调电压、失调电流温漂小
5. **通用性强**：应用广泛

#### 10.1.2 电路符号

```
           ┌─────┐
    v+ ────┤     │
           │  ∇  ├─── v_o
    v- ────┤     │
           └─────┘
     +    -       out
    同相 反相
```

**主要参数**：

| 参数 | 符号 | 典型值 |
|------|------|--------|
| 开环增益 | $A_{vo}$ | 10⁵~10⁶ |
| 输入失调电压 | $V_{IO}$ | mV级 |
| 输入偏置电流 | $I_{IB}$ | nA~pA级 |
| 输入电阻 | $r_i$ | MΩ~GΩ级 |
| 输出电阻 | $r_o$ | <100Ω |
| 共模抑制比 | $K_{CMR}$ | >100dB |
| 转换速率 | $S_R$ | V/μs级 |

### 10.2 理想运放特性

#### 10.2.1 理想运放的两个重要特性

1. **虚短**：$v_+ \approx v_-$（两输入端电位近似相等）
2. **虚断**：$i_+ \approx i_- \approx 0$（两输入端电流近似为零）

这两个特性是分析运放应用电路的基础。

### 10.3 运放的开环与闭环

#### 10.3.1 开环应用

```
        v_i
         │
         ├───(+)──┤
         │         │ A_vo
    ┌────┴────┐    │
    │    ∇    ├────┘
    └────┬────┘    │
         │         │
         └────○────┘
```

- $A_{vo}$ 很大，输入端差模电压极小
- 任何微小的输入都会使输出饱和
- 实际用于比较器

#### 10.3.2 闭环应用

通过负反馈网络，将输出信号引回输入端，控制增益。

### 10.4 集成运放的内部结构

#### 10.4.1 典型结构

- **输入级**：差分放大电路（提供高输入阻抗和共模抑制）
- **中间级**：多级共射放大（提供高增益）
- **输出级**：互补对称功放（低输出阻抗）
- **偏置电路**：恒流源电路（提供稳定偏置）

#### 10.4.2 常见集成运放型号

| 型号 | 类型 | 特点 |
|------|------|------|
| μA741 | 通用型 | 经典运放，应用广泛 |
| LM358 | 低功耗 | 单电源供电 |
| OP07 | 高精度 | 低失调电压 |
| NE5532 | 低噪声 | 音频应用 |
| TL084 | JFET输入 | 高输入阻抗 |

## 第11章 负反馈技术

### 11.1 反馈的基本概念

#### 11.1.1 反馈的定义

将放大电路输出信号的一部分或全部，通过反馈网络送回输入端，与输入信号比较（加或减），这个过程称为**反馈**。

#### 11.1.2 反馈的分类

**按反馈极性分**：

| 类型 | 定义 | 效果 |
|------|------|------|
| 正反馈 | 反馈信号增强净输入 | 增益提高，稳定性变差 |
| 负反馈 | 反馈信号削弱净输入 | 增益降低，稳定性提高 |

**按采样方式分**：

| 类型 | 采样对象 | 输出端连接方式 |
|------|----------|----------------|
| 电压反馈 | 输出电压 | 与输出端并联 |
| 电流反馈 | 输出电流 | 与输出端串联 |

**按比较方式分**：

| 类型 | 输入端连接方式 |
|------|----------------|
| 串联反馈 | 与输入信号串联 |
| 并联反馈 | 与输入信号并联 |

### 11.2 负反馈的四种组态

#### 11.2.1 电压串联负反馈

```
        v_i
         │
    ┌────┴───┐
    │        │  v_f
    ├─(+)────┼─────────┐
    │        │         │
    │    ┌───┴───┐     │
    │    │  放大  │     │
    │    │   A   │     │
    │    └───┬───┘     │
    │        │         │
    └────────┼─────────┘
             │
         反馈网络 F
```

**特点**：
- 采样：电压
- 比较：串联
- $v_f$ 与 $v_i$ 串联后加到放大器输入
- 输入电阻增加
- 输出电阻减小

#### 11.2.2 电压并联负反馈

**特点**：
- 采样：电压
- 比较：并联
- 反馈信号与输入信号在输入端并联

#### 11.2.3 电流串联负反馈

**特点**：
- 采样：电流
- 比较：串联
- 反馈电压与输入电压串联

#### 11.2.4 电流并联负反馈

**特点**：
- 采样：电流
- 比较：并联
- 反馈信号与输入信号在输入端并联

### 11.3 负反馈对放大电路性能的影响

#### 11.3.1 增益

闭环增益：
$$A_f = \frac{A}{1+AF}$$

其中 $1+AF$ 称为**反馈深度**。

当 $|1+AF| \gg 1$（深负反馈）时：
$$A_f \approx \frac{1}{F}$$

闭环增益与有源器件参数几乎无关，仅由反馈网络决定。

#### 11.3.2 稳定性的改善

负反馈提高了增益的稳定性：

$$\frac{dA_f}{A_f} = \frac{1}{1+AF} \cdot \frac{dA}{A}$$

当 $1+AF$ 增大时，增益变化减小。

#### 11.3.3 扩展通频带

负反馈将上限频率提高 $(1+AF)$ 倍，下限频率降低 $(1+AF)$ 倍：

$$f_{hf} = (1+AF)f_h$$
$$f_{lf} = \frac{f_l}{1+AF}$$

#### 11.3.4 减小非线性失真

负反馈可以减小由放大器非线性引起的失真，但只能减小放大器自身产生的失真，对输入信号本身的失真无能为力。

#### 11.3.5 对输入/输出电阻的影响

| 反馈类型 | 输入电阻 | 输出电阻 |
|----------|----------|----------|
| 电压负反馈 | ↑ | ↓ |
| 电流负反馈 | ↑ | ↑ |
| 串联负反馈 | ↑ | - |
| 并联负反馈 | ↓ | - |

### 11.4 负反馈放大电路的自激振荡

#### 11.4.1 自激条件

当 $|1+AF| = 0$ 时，即使无输入信号，也有输出：

$$AF = -1$$

或：
- $|AF| = 1$（幅度条件）
- $\varphi_{AF} = \pm180°$（相位条件）

#### 11.4.2 稳定判据

稳定条件：环路增益 $|AF|$ 在达到0dB时，其相位裕度 $>45°$。

## 第12章 集成运放的应用电路

### 12.1 比例运算电路

#### 12.1.1 反相比例运算电路

```
        R_1
 vi ○──┤├──┬──○── vo
             │
           ┌─┴─┐
           │   │
           │ - │◄──反馈
           │   │
           └─┬─┘
             │
            R_F
             │
             ── GND
```

**电路分析**：
- 根据"虚短"：$v_+ = v_- = 0$（虚地）
- 根据"虚断"：$i_1 = i_F$
- $\frac{v_i - 0}{R_1} = \frac{0 - v_o}{R_F}$

**电压增益**：
$$A_{vf} = \frac{v_o}{v_i} = -\frac{R_F}{R_1}$$

**输入电阻**：
$$r_i = R_1$$

**特点**：
- $v_o$ 与 $v_i$ 反相
- 输入电阻等于 $R_1$

#### 12.1.2 同相比例运算电路

```
 vi ○──┬───○── vo
        │   │
       R_1 │   │
        └───┤- │
            │   │◄──反馈
           +│   │
            └───┤
               R_F
                │
                ── GND
```

**电压增益**：
$$A_{vf} = 1 + \frac{R_F}{R_1}$$

**特点**：
- $v_o$ 与 $v_i$ 同相
- 输入电阻极高
- 当 $R_F = 0, R_1 = \infty$ 时，$A_{vf} = 1$，成为**电压跟随器**

### 12.2 加法运算电路

#### 12.2.1 反相加法器

```
 v1○──R_1─┐
          │
 v2○──R_2─┼──┬──○── vo
          │  │
 v3○──R_3─┘  │
              │
            ┌─┴─┐
            │ - │
            └─┬─┘
              │
             R_F
              │
              ── GND
```

**输出电压**：
$$v_o = -\left(\frac{R_F}{R_1}v_1 + \frac{R_F}{R_2}v_2 + \frac{R_F}{R_3}v_3\right)$$

当 $R_1 = R_2 = R_3 = R$ 时：
$$v_o = -\frac{R_F}{R}(v_1 + v_2 + v_3)$$

#### 12.2.2 同相加法器

各输入信号加到同相端。

### 12.3 减法运算电路（差分放大器）

```
 v1○──R_1─┬──┬──○── vo
          │  │
          │ ┌┴─┐
          │ │  │
          │ │ - │◄──反馈
          │ └┬─┘
          │  │
 v2○──R_2─┘  │
             R_F
              │
              ── GND
```

**输出电压**：
$$v_o = \frac{R_F}{R_1}v_1 - \frac{R_F}{R_2}v_2$$

当 $R_1 = R_2 = R_F$ 时：
$$v_o = v_1 - v_2$$

### 12.4 积分运算电路

```
 vi ○──R_1─┬──┬──○── vo
              │
            ┌─┴─┐
            │   │
            │ - │◄──反馈
            │   │ C_F
            └─┬─┼───┘
              │  │
              │  ── C_F
              │  │
              └──┘
```

**输出电压**：
$$v_o = -\frac{1}{R_1C_F}\int v_i \, dt$$

### 12.5 微分运算电路

```
 vi ○──C_1─┬──┬──○── vo
              │
            ┌─┴─┐
            │   │
            │ - │◄──反馈
            │   │ R_F
            └─┬─┤
              │  │
              │  R_F
              │  │
              └──┘
```

**输出电压**：
$$v_o = -R_F C_1 \frac{dv_i}{dt}$$

## 第13章 直流稳压电源

### 13.1 直流稳压电源的组成

```
交流输入 → 变压器 → 整流电路 → 滤波电路 → 稳压电路 → 直流输出
```

| 环节 | 功能 |
|------|------|
| 变压器 | 电压变换（AC→AC） |
| 整流 | 将交流变为脉动直流 |
| 滤波 | 减小输出纹波 |
| 稳压 | 稳定输出电压 |

### 13.2 整流电路

#### 13.2.1 单相桥式整流

已在第3章详细讨论，输出电压平均值：

$$V_{o(AV)} = 0.9V_2$$

### 13.3 滤波电路

#### 13.3.1 电容滤波

**滤波原理**：
- 利用电容的充放电平滑输出电压
- 电容两端电压不能突变

**输出特性**：
$$V_{o(AV)} \approx \begin{cases} 1.2V_2 & \text{空载} \\ 1.0V_2 & \text{轻载} \\ 0.9V_2 & \text{重载} \end{cases}$$

**电容选择**：
$$C \geq \frac{(3\sim5)T}{R_L}$$

其中 $T$ 为交流电源周期。

#### 13.3.2 电感滤波

适用于大电流负载，输出特性硬。

#### 13.3.3 π型滤波

```
    L
 ──┤├──┐
         │
 ──┤├──┤
         ├──○── Vo
 ──┤├──┤
         │
    C₁   C₂
```

滤波效果最好，但体积大、成本高。

### 13.4 稳压管稳压电路

#### 13.4.1 电路结构

```
       R         R_L
Vi ○──┤├──┬──○──┤├────○── Vo
           │        │
           │        │
           └──┤├─│←─┘
              ││
             D_Z
              │
              ── GND
```

#### 13.4.2 工作原理

稳压管 $D_Z$ 工作在反向击穿状态，其两端电压 $V_Z$ 基本恒定。

**稳压条件**：
$$I_Z_{min} < I_Z < I_Z_{max}$$

**限流电阻**：
$$R = \frac{V_i - V_Z}{I_Z + I_L}$$

#### 13.4.3 电路特点

- 电路简单
- 输出电压固定（由稳压管决定）
- 负载电流不能太大
- 稳压精度不高

### 13.5 串联型稳压电路

#### 13.5.1 电路结构

```
         R1
Vi ○──┤├──┬──○── Vo
           │  │
           │  ├──○── R_L
           │  │
          ┌┴──┴┐
          │ 调整管 │
          │   T   │
          └──┬──┘
             │
           ┌─┴─┐
           │   │ ← 采样
        +──│比较│───│──○──
           │放大│
        -──└─┬─┘
             │
             │ Vref
           D_Z
```

#### 13.5.2 稳压原理

输出电压变化经采样、放大后控制调整管，使输出电压保持稳定：

$$\Delta V_o \uparrow \rightarrow V_{B} \uparrow \rightarrow V_{CE} \uparrow \rightarrow V_o \downarrow$$

#### 13.5.3 输出电压

$$V_o = V_Z \cdot \frac{R_1 + R_2}{R_2}$$

调节 $R_1$ 或 $R_2$ 可改变输出电压。

### 13.6 集成稳压器

#### 13.6.1 三端集成稳压器

| 类型 | 型号 | 输出电压 |
|------|------|----------|
| 固定正压 | 78XX | +5V, +9V, +12V, +15V, +18V, +24V |
| 固定负压 | 79XX | -5V, -9V, -12V, -15V, -18V, -24V |
| 可调正压 | LM317 | 1.25V~37V |
| 可调负压 | LM337 | -1.25V~-37V |

#### 13.6.2 典型应用

```
         Vin                Vout
  ○──────┬────○──────────┬─────○
         │                 │
        C1                 C2
         │                 │
    ┌────┴────────────────┴────┐
    │         78XX               │
    │  IN          OUT   GND    │
    └────┬────────────────┬─────┘
         │                 │
         └────○───────────┘
              GND
```

## 第14章 信号产生电路

### 14.1 正弦波振荡的条件

#### 14.1.1 起振条件

$$\begin{cases} |AF| > 1 & \text{起振条件} \\ \varphi_{AF} = 2n\pi & (n=0,1,2...) \end{cases}$$

#### 14.1.2 平衡条件（稳幅振荡）

$$\begin{cases} |AF| = 1 & \text{幅值平衡} \\ \varphi_{AF} = 2n\pi & \text{相位平衡} \end{cases}$$

### 14.2 RC正弦波振荡器

#### 14.2.1 RC桥式振荡器（文氏桥振荡器）

```
        R_F
       ┌┴┐
       │ │ ───┐
       └┬┘   │  +
    Vi○┤ ├──○┤  │─○ Vo
       └┬┘   │  -
       │ │ ───┘
       └─┘
        R
        C
```

**选频网络**：RC串并联网络
- 谐振频率：$f_0 = \frac{1}{2\pi RC}$
- 相位变化：0°~180°

**振荡频率**：
$$f_0 = \frac{1}{2\pi RC}$$

**起振条件**：
$$A_v = 1 + \frac{R_F}{R} > 3$$

### 14.3 LC正弦波振荡器

#### 14.3.1 变压器反馈式振荡器

利用LC谐振回路选频，变压器耦合反馈。

**振荡频率**：
$$f_0 = \frac{1}{2\pi\sqrt{LC}}$$

#### 14.3.2 电感三点式振荡器（哈特莱振荡器）

```
        L₁       L₂
       ┌┴─┴─┐   ┌┴─┴─┐
       │    │   │    │
       └┬───┘   └───┬┘
         │         │
         └─C─┬─C─┘
             │
            GND
```

**振荡频率**：
$$f_0 = \frac{1}{2\pi\sqrt{(L_1+L_2) \cdot C}}$$

#### 14.3.3 电容三点式振荡器（考毕兹振荡器）

**振荡频率**：
$$f_0 = \frac{1}{2\pi\sqrt{L \cdot \frac{C_1C_2}{C_1+C_2}}}$$

### 14.4 石英晶体振荡器

#### 14.4.1 石英晶体的压电效应

- 压电效应：机械振动 ↔ 电振荡
- 谐振频率稳定，$Q$值极高（10⁴~10⁶）

#### 14.4.2 等效电路与阻抗特性

```
        C₁
       ┌┴┐
       │ │   L
       │ ├──┤├──┐
       └┬┘  │  │
        │   └──┘
       C₂   │
        │   GND
        └───┘
```

- **串联谐振**：$f_s = \frac{1}{2\pi\sqrt{LC}}$，阻抗最小
- **并联谐振**：$f_p = \frac{1}{2\pi\sqrt{L\frac{CC_1}{C+C_1}}}$，阻抗最大

#### 14.4.3 应用

- 串联型晶体振荡器：工作于 $f_s$
- 并联型晶体振荡器：工作于 $f_s$ 与 $f_p$ 之间

## 第15章 滤波电路

### 15.1 滤波电路的基本概念

#### 15.1.1 滤波器的分类

| 类型 | 通带 | 阻带 |
|------|------|------|
| 低通滤波器(LPF) | 0 ~ $f_c$ | $f_c$ ~ ∞ |
| 高通滤波器(HPF) | $f_c$ ~ ∞ | 0 ~ $f_c$ |
| 带通滤波器(BPF) | $f_{L}$ ~ $f_{H}$ | 两端 |
| 带阻滤波器(BEF) | 两端 | $f_{L}$ ~ $f_{H}$ |

#### 15.1.2 主要参数

- **通带增益**：通带内电压放大倍数
- **截止频率**：增益下降到通带增益的 $1/\sqrt{2}$ 处
- **过渡带宽**：通带到阻带的过渡宽度

### 15.2 一阶RC滤波电路

#### 15.2.1 一阶低通滤波器

```
    R ──┬──┬──○── Vo
        │  │
        C  R_L
        │  │
       GND │
        └──┘
```

**传递函数**：
$$A(j\omega) = \frac{1}{1+j\omega RC}$$

**幅频特性**：
$$|A| = \frac{1}{\sqrt{1+(\omega RC)^2}}$$

**截止频率**：
$$f_c = \frac{1}{2\pi RC}$$

#### 15.2.2 一阶高通滤波器

```
    C ──┬──┬──○── Vo
        │  │
        R  R_L
        │  │
       GND │
        └──┘
```

**传递函数**：
$$A(j\omega) = \frac{j\omega RC}{1+j\omega RC}$$

### 15.3 二阶滤波电路

#### 15.3.1 二阶低通滤波器

```
      R₁      C₁
 vi ○──┤├──┬──○──┤├──┬──○── Vo
            │      │   │
            │      │   │
           R_F     │   │
            │      │   │
            └──────┴───┘
                 │
                C₂
                 │
                GND
```

**特点**：
- 滤波效果比一阶好
- -40dB/decade 下降斜率
- 可通过改变元件值调节 $f_c$

### 15.4 有源滤波电路

使用集成运放实现滤波，具有输入阻抗高、输出阻抗低、可放大信号等优点。

#### 15.4.1 压控电压源滤波器

- 运放接成同相放大
- $Q$ 值由反馈网络决定
- 应用广泛

#### 15.4.2 无限增益多路反馈滤波器

- 运放工作于反相
- 多路反馈
- 适合高阶滤波器

### 附录B 常用参数表

**硅/锗二极管参数**：

| 参数 | 硅管 | 锗管 |
|------|------|------|
| 正向压降 $V_F$ | 0.6~0.8V | 0.2~0.3V |
| 反向饱和电流 $I_S$ | <1μA | <100μA |
| 最高工作频率 | 较高 | 较低 |

**热电压**：
$$V_T = \frac{kT}{q} \approx 26\text{mV} \quad (T=300K)$$
