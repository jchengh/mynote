# 第一篇 半导体器件基础

# 第1章 半导体物理基础

## 1.1 半导体的物理特性

### 1.1.1 半导体的基本概念

半导体是指电阻率介于导体和绝缘体之间的材料，其电阻率大约在10⁻³~10⁸ Ω·cm的范围内。常用的半导体材料包括：

| 材料 | 禁带宽度 Eg | 应用 |
|------|-------------|------|
| 硅(Si) | 1.12 eV | 最广泛使用的半导体材料 |
| 锗(Ge) | 0.67 eV | 早期使用的半导体材料 |
| 砷化镓(GaAs) | 1.43 eV | 高频、微波、光电应用 |

半导体具有以下几个独特的物理特性：

1. **热敏性**：半导体的导电能力随温度升高而显著增强，这与金属导体截然相反。
2. **光敏性**：半导体的导电能力受光照影响，某些半导体在光照下电阻急剧减小。
3. **掺杂特性**：在纯净半导体中掺入微量杂质元素，可使其导电能力发生显著变化。

### 1.1.2 本征半导体

**本征半导体**：完全纯净、结构完整的半导体晶体。在绝对零度(T=0K)时，本征半导体的价电子被牢牢束缚在共价键中，不能自由移动，因此不导电。

当温度升高或受到光照时，部分价电子获得足够能量挣脱共价键的束缚，成为**自由电子**，同时在共价键中留下一个空位，这个空位称为**空穴**。

```
共价键结构示意图：
    
    Si ── Si ── Si
    │         │
    Si       Si
    │         │
    Si ── Si ── Si
    
温度升高时：Si → Si⁺ + e⁻（产生电子-空穴对）
                     ↑    ↑
                   空穴  自由电子
```

### 1.1.3 载流子的产生与复合

在本征半导体中，热激发会产生**电子-空穴对**，这个过程称为**本征激发**。同时，自由电子在运动过程中可能与空穴相遇，重新被空穴捕获，这个过程称为**复合**。

在热平衡状态下，产生率等于复合率，电子浓度和空穴浓度相等：

$$n_i = p_i = A \cdot T^{3/2} \cdot e^{-E_g/(2kT})$$

其中：
- $n_i$：本征电子浓度（cm⁻³）
- $p_i$：本征空穴浓度（cm⁻³）
- $A$：与半导体材料有关的常数（硅约为5.2×10¹⁵ cm⁻³·K⁻³/²）
- $T$：热力学温度（K）
- $E_g$：禁带宽度（eV）
- $k$：玻尔兹曼常数（8.62×10⁻⁵ eV/K）

**室温(300K)下的本征载流子浓度**：
- 硅(Si)：$n_i \approx 1.5 \times 10^{10}$ cm⁻³
- 锗(Ge)：$n_i \approx 2.4 \times 10^{13}$ cm⁻³

这意味着在1 cm³的硅晶体中，大约有10万亿个电子-空穴对。相比之下，硅的原子密度约为5×10²² cm⁻³，所以本征硅中自由载流子浓度约为原子浓度的万亿分之一。

## 1.2 杂质半导体

### 1.2.1 杂质半导体的形成原理

通过在纯净半导体中掺入特定元素（称为**掺杂**），可以显著改变半导体的导电类型和载流子浓度。这种通过掺杂形成的半导体称为**杂质半导体**。

### 1.2.2 N型半导体

在硅（或锗）晶体中掺入**五价元素**（如磷P、砷As、锑Sb），五价元素的原子有5个价电子。当它替代硅原子进入晶格时，4个电子与周围的4个硅原子形成共价键，多余的1个电子很容易挣脱原子核的束缚成为自由电子。

```
磷原子在硅晶体中的掺杂示意图：

    Si ── Si ── P ── Si
          ↑         
    多余的自由电子(e⁻)
    
施主杂质电离：P → P⁺ + e⁻
```

**特点**：
- 多子（多数载流子）：电子
- 少子（少数载流子）：空穴
- 掺杂浓度 $N_D \gg n_i$
- 电子浓度 $n_n \approx N_D$
- 空穴浓度 $p_n = \frac{n_i^2}{N_D}$

**重要公式**：

在N型半导体中，质量作用定律仍然成立：
$$n_n \cdot p_n = n_i^2$$

由于电子是主要载流子，这种半导体称为**电子型半导体**或**N型半导体**。

### 1.2.3 P型半导体

在硅晶体中掺入**三价元素**（如硼B、镓Ga、铟In），三价元素的原子只有3个价电子。当它替代硅原子进入晶格时，只能与3个硅原子形成完整的共价键，在与第4个硅原子之间留下一个空位（空穴）。

```
硼原子在硅晶体中的掺杂示意图：

    Si ── Si ── B ── Si
          ↑         
    产生的空穴(h⁺)
    
受主杂质电离：B + e⁻ → B⁻ + h⁺
```

**特点**：
- 多子：空穴
- 少子：电子
- 掺杂浓度 $N_A \gg n_i$
- 空穴浓度 $p_p \approx N_A$
- 电子浓度 $n_p = \frac{n_i^2}{N_A}$

### 1.2.4 杂质浓度的估算

室温下，杂质原子几乎全部电离。假设完全电离，则：

- N型：$n_n \approx N_D$
- P型：$p_p \approx N_A$

典型的掺杂浓度范围：
- 轻掺杂：$10^{14}$ ~ $10^{15}$ cm⁻³
- 中掺杂：$10^{16}$ ~ $10^{18}$ cm⁻³
- 重掺杂：$10^{19}$ ~ $10^{20}$ cm⁻³

## 1.3 载流子的运动

### 1.3.1 漂移运动

在**电场**作用下，载流子会发生定向移动，这种运动称为**漂移运动**。

```
漂移运动示意图：
    
    电场 E ──→──
    
    电子：←───e⁻───← (逆电场方向)
    空穴：───h⁺──→ (顺电场方向)
    
    电流方向相同（向下）
```

**漂移电流密度**：

$$J_{drift} = q(n\mu_n + p\mu_p)E = \sigma E$$

其中：
- $q = 1.602 \times 10^{-19}$ C（电子电荷）
- $\mu_n$：电子迁移率（cm²/V·s）
- $\mu_p$：空穴迁移率（cm²/V·s）
- $\sigma$：电导率（S/cm）

**室温下硅的迁移率**：
| 载流子 | 迁移率 $\mu$ |
|--------|-------------|
| 电子 | 1350 cm²/(V·s) |
| 空穴 | 500 cm²/(V·s) |

### 1.3.2 扩散运动

当半导体中**载流子浓度分布不均匀**时，载流子会从高浓度区向低浓度区扩散，这种运动称为**扩散运动**。

```
扩散运动示意图：

浓度 n ─→→→→→  →→→→→→→→
分布：  ████████████████→ x
        高浓度      低浓度
        
        ←── 扩散方向 ──→
```

**扩散电流密度**：

扩散电流由浓度梯度引起：

$$J_n = qD_n \frac{dn}{dx}$$
$$J_p = -qD_p \frac{dp}{dx}$$

其中：
- $D_n$：电子扩散系数（cm²/s）
- $D_p$：空穴扩散系数（cm²/s）
- 负号表示空穴电流方向与浓度梯度方向相同

**爱因斯坦关系**：

漂移和扩散是载流子运动的两种形式，它们之间存在内在联系：

$$\frac{D_n}{\mu_n} = \frac{D_p}{\mu_p} = \frac{kT}{q} = V_T$$

其中 $V_T$ 称为**热电压**，在室温（300K）下：

$$V_T = \frac{kT}{q} \approx \frac{1.38 \times 10^{-23} \times 300}{1.602 \times 10^{-19}} \approx 26 \text{ mV}$$

### 1.3.3 总电流

在半导体中，漂移电流和扩散电流同时存在。总电流密度为：

$$J = J_{drift} + J_{diff} = q(n\mu_n + p\mu_p)E + q(D_n \frac{dn}{dx} - D_p \frac{dp}{dx})$$

## 1.4 复合与产生

### 1.4.1 复合机制

载流子的复合是导带中的电子回到价带，与空穴复合消失的过程。复合可以通过多种机制发生：

1. **直接复合**：电子直接从导带跃迁回价带，与空穴直接复合
2. **间接复合（SRH复合）**：通过复合中心（如杂质或缺陷）进行复合
3. **表面复合**：在半导体表面发生的复合

### 1.4.2 少数载流子的寿命

**少子寿命** $\tau$ 定义为：少子从产生到复合所经历的平均时间。

少子寿命是表征半导体材料质量的重要参数：
- 硅的少子寿命：微秒到毫秒级
- 少子寿命越长，器件性能越好

**少子扩散长度**：

$$L = \sqrt{D\tau}$$

少子扩散长度表示少子在复合前平均扩散的距离。

## 1.5 温度对半导体的影响

### 1.5.1 温度对载流子浓度的影响

温度对半导体载流子浓度的影响非常显著：

$$n_i = A \cdot T^{3/2} \cdot e^{-E_g/(2kT)}$$

温度每升高8~10°C，本征载流子浓度大约增加一倍。

### 1.5.2 温度对迁移率的影响

温度升高时，晶格振动加剧，载流子与晶格的散射增强，导致迁移率下降。

$$\mu \propto T^{-m}$$

其中 $m \approx 2.5$（硅中电子）或 $m \approx 2.3$（硅中空穴）。

### 1.5.3 温度对电阻率的影响

杂质半导体的电阻率随温度变化：

$$\rho = \frac{1}{q(n\mu_n + p\mu_p)}$$

在杂质浓度一定时，电阻率随温度的变化比较复杂。

# 第2章 PN结理论

## 2.1 PN结的形成

### 2.1.1 载流子的扩散

当P型半导体和N型半导体接触形成PN结时，由于两侧多子浓度存在巨大差异：

- P区：空穴浓度 $p_p \approx N_A$（高），电子浓度 $n_p = n_i^2/N_A$（低）
- N区：电子浓度 $n_n \approx N_D$（高），空穴浓度 $p_n = n_i^2/N_D$（低）

浓度差导致**扩散运动**：

```
PN结形成过程：

P区 ←────────────────→ N区
[空穴]  ────→  [空穴]
(高浓度)        (低浓度)

[电子]  ←───  [电子]
(低浓度)        (高浓度)

扩散后：
P区                    N区
────────────────┬───────
 负离子区      │      正离子区
────────────────┼───────
  空间电荷区（耗尽层）
```

### 2.1.2 空间电荷区的形成

扩散运动的结果：

1. **P区失去空穴**：靠近界面的空穴扩散到N区，与电子复合消失，留下带负电的电离受主杂质离子 $N_A^-$
2. **N区失去电子**：靠近界面的电子扩散到P区，与空穴复合消失，留下带正电的电离施主杂质离子 $N_D^+$

这些不能移动的杂质离子形成**空间电荷区**，也称为**耗尽层**或**阻挡层**。

### 2.1.3 内建电场与内建电位

空间电荷区形成后，产生**内建电场**（或称为**势垒电场**），方向由N区指向P区。内建电场：

1. **阻碍多子扩散**：使多子扩散减弱
2. **促进少子漂移**：使P区和N区的少子各自向对方漂移

最终，扩散作用与漂移作用达到**动态平衡**，此时：

$$I_{扩散} + I_{漂移} = 0$$

即净电流为零。

**内建电位**（或称为**接触电位差**、**势垒电位**）：

$$V_{bi} = V_T \ln\left(\frac{N_AN_D}{n_i^2}\right)$$

典型值：
- 硅PN结：$V_{bi} \approx 0.6$ ~ $0.8$ V
- 锗PN结：$V_{bi} \approx 0.2$ ~ $0.3$ V

### 2.1.4 空间电荷区的宽度

假设突变结近似，耗尽层宽度：

$$W = \sqrt{\frac{2\varepsilon_s(V_{bi} - V)}{q}\left(\frac{1}{N_A} + \frac{1}{N_D}\right)}$$

其中：
- $\varepsilon_s$：半导体介电常数（硅 $\varepsilon_s = 11.7\varepsilon_0$）
- $V$：外加电压（正偏时为正，反偏时为负）
- $N_A, N_D$：受主和施主浓度

**特点**：
- 两侧浓度相等时，耗尽层在两侧是对称的
- 一侧浓度远大于另一侧时，耗尽层主要分布在轻掺杂一侧
- 反偏电压增大时，耗尽层变宽

## 2.2 PN结的单向导电性

### 2.2.1 外加正向电压

当P区接高电位、N区接低电位时，称为**正向偏置**（Forward Bias）。

```
正向偏置电路：

    +  ──→── [P]──[N]───  -
   V_F     ↓           ↓
    │      ↓           ↓
    └──────┴───────────┘
```

**物理过程**：

1. 外加电场方向与内建电场方向**相反**
2. 外加电场**削弱**内建电场
3. 势垒降低，耗尽层变窄
4. 多子扩散增强，形成**正向电流** $I_F$
5. 电源不断向半导体注入载流子，维持电流

**势垒变化**：

$$V_{bi}' = V_{bi} - V_F$$

**正向电流方程**（肖克利二极管方程）：

$$I_F = I_S \left( e^{\frac{V_F}{V_T}} - 1 \right)$$

其中：
- $I_S$：反向饱和电流（硅管约 $10^{-15}$ ~ $10^{-12}$ A）
- $V_T$：热电压（约26 mV @ 300K）

**正向特性的特点**：

1. 当 $V_F < V_{th}$（门坎电压，约0.5V）时，电流很小
2. 当 $V_F \approx 0.6$ ~ $0.8$ V（硅管）时，电流明显增长
3. 电流随电压指数增加

### 2.2.2 外加反向电压

当P区接低电位、N区接高电位时，称为**反向偏置**（Reverse Bias）。

```
反向偏置电路：

    -  ──→── [N]──[P]───  +
   V_R     ↓           ↓
    │      ↓           ↓
    └──────┴───────────┘
```

**物理过程**：

1. 外加电场方向与内建电场方向**相同**
2. 外加电场**增强**内建电场
3. 势垒升高，耗尽层变宽
4. 扩散几乎停止
5. 少子漂移形成**反向电流** $I_R$

**反向电流的特点**：

1. 数值很小（μA级或更小）
2. 基本不随反向电压增大而变化（达到饱和）
3. 受温度影响很大：温度每升高10°C，反向电流约增加一倍

**理想情况下**：
$$I_R \approx -I_S$$

## 2.3 PN结的电流电压方程

### 2.3.1 肖克利方程

完整的PN结电流方程：

$$I = I_S \left( e^{\frac{V}{V_T}} - 1 \right)$$

**指数区（正偏）**：当 $V \gg V_T$ 时：
$$I \approx I_S e^{\frac{V}{V_T}}$$

**反偏区**：当 $V < 0, |V| \gg V_T$ 时：
$$I \approx -I_S$$

**零偏置**：$V = 0$ 时，$I = 0$

### 2.3.2 反向饱和电流

反向饱和电流 $I_S$ 的表达式：

$$I_S = qA\left(\frac{D_p}{L_p} p_{n0} + \frac{D_n}{L_n} n_{p0}\right) = qA\left(\frac{D_p}{L_p}\frac{n_i^2}{N_D} + \frac{D_n}{L_n}\frac{n_i^2}{N_A}\right)$$

其中：
- $A$：结面积
- $D_n, D_p$：电子、空穴扩散系数
- $L_n, L_p$：电子、空穴扩散长度

**影响 $I_S$ 的因素**：

1. **温度**：$I_S$ 随温度升高而急剧增大（因为 $n_i \propto T^{3/2}e^{-E_g/2kT}$）
2. **禁带宽度**：禁带宽度越大，$I_S$ 越小（硅 < 锗）
3. **掺杂浓度**：掺杂浓度越高，$I_S$ 越小
4. **少子寿命**：少子寿命越长，$L$ 越大，$I_S$ 越小

## 2.4 PN结的击穿特性

### 2.4.1 击穿现象

当反向电压增大到某一值时，反向电流急剧增加，这种现象称为**击穿**。相应的电压称为**击穿电压** $V_{BR}$。

### 2.4.2 雪崩击穿

**原理**：当反向电压足够高时，载流子在耗尽层中获得足够的能量，在与原子碰撞时能够使共价键断裂，产生新的电子-空穴对。新产生的载流子又加速获得能量，再次碰撞产生更多载流子，形成连锁反应。

```
雪崩倍增示意图：

    ←── e⁻ ←── e⁻ ←──
    
    █   █   █   █   ← 原子
    ↓
    ↓ ←───── e⁻ ←───── ←─ 碰撞电离
    ↓           ↓
    █   █   █   █
    
    载流子数量呈"雪崩"式增长
```

**特点**：
- 发生在掺杂浓度**较低**的区域
- 击穿电压较高（通常大于6V）
- 击穿电压具有**正温度系数**（温度升高，击穿电压升高）
- 雪崩击穿是碰撞电离的结果

### 2.4.3 齐纳击穿

**原理**：当PN结两侧掺杂浓度很高时，耗尽层很窄，反向电压在耗尽层中产生很强的电场（可达10⁶ V/cm）。强电场可以直接将共价键中的电子拉出来，产生电子-空穴对，这就是**场致激发**或**隧穿效应**。

**特点**：
- 发生在掺杂浓度**很高**的区域
- 击穿电压较低（通常低于5V）
- 击穿电压具有**负温度系数**（温度升高，击穿电压降低）
- 齐纳击穿是量子隧穿的结果

### 2.4.4 热击穿

如果雪崩击穿或齐纳击穿发生在局部区域，导致局部温度过高，可能引发**热击穿**，造成器件永久损坏。

**防止措施**：
- 串联限流电阻
- 改善散热条件
- 避免超过最大功耗

### 2.4.5 两种击穿机理的分界

通常以 $V_{BR} = 5$ ~ $6$ V 作为分界：
- $V_{BR} < 4$ ~ $5$ V：以齐纳击穿为主
- $V_{BR} > 6$ ~ $7$ V：以雪崩击穿为主
- $V_{BR} \approx 5$ ~ $6$ V：两种机理并存

## 2.5 PN结的电容效应

### 2.5.1 势垒电容（结电容）

空间电荷区类似于一个电容器，其电容称为**势垒电容**或**结电容**。

```
势垒电容模型：

    P区 ──┬───────────┬── N区
          │   耗尽层   │
          │  (绝缘层)  │
          └───────────┘
               ↑
            等效电容
```

**势垒电容公式**（突变结近似）：

$$C_T = \frac{\varepsilon_s A}{W} = A\sqrt{\frac{q\varepsilon_s}{2(V_{bi} - V)}\frac{N_AN_D}{N_A + N_D}}$$

**特点**：
- 与外加电压有关
- 反偏时 $C_T$ 较小，随反偏电压增大而减小
- 正偏时 $C_T$ 较大

**变容二极管**：利用势垒电容随电压变化的特性，可作为压控电容器使用。

### 2.5.2 扩散电容

当PN结正向偏置时，注入到对方的少子在扩散区内积累，形成**扩散电容** $C_D$。

**扩散电容公式**：

$$C_D = \frac{qI}{V_T}\tau$$

其中 $\tau$ 是少子渡越时间。

**特点**：
- 只在正偏时显著
- 与正向电流成正比
- 通常远大于势垒电容

### 2.5.3 结电容的影响

结电容的存在限制了PN结的高频性能：

$$f_T = \frac{g_m}{2\pi C_\pi}$$

其中 $f_T$ 称为**特征频率**，是使电流增益降为1的频率。

# 第3章 半导体二极管

## 3.1 二极管的结构与分类

### 3.1.1 基本结构

半导体二极管由一个PN结构成，具有两个电极：
- **阳极（A）**：P型区
- **阴极（K）**：N型区

```
二极管电路符号：

    A ──▶┃── K
    阳极  阴极
    
    正向：电流从A流向K
    反向：电流从K流向A
```

### 3.1.2 二极管的分类

| 分类方式 | 类型 | 特点 |
|----------|------|------|
| 按材料 | 硅二极管 | 正向压降0.6~0.8V |
|          | 锗二极管 | 正向压降0.2~0.3V |
| 按结构 | 点接触型 | 结电容小，适用于高频 |
|          | 面接触型 | 结电容大，适用于低频大电流 |
|          | 平面型 | 性能介于两者之间 |
| 按用途 | 整流二极管 | 电源整流 |
|          | 检波二极管 | 信号检波 |
|          | 稳压二极管 | 稳压 |
|          | 变容二极管 | 压控电容 |
|          | 发光二极管 | 光电转换 |
|          | 光电二极管 | 光电探测 |
|          | 肖特基二极管 | 低正向压降 |
|          | 齐纳二极管 | 稳压 |

## 3.2 二极管的伏安特性

### 3.2.1 正向特性

```
正向特性曲线：

        I (mA)
        ↑
    100 ┤           ●●
        │        ●
     50 ┤       ●
        │     ●
     10 ┤    ●
        │   ●
      1 ┤  ●
        │ ●
      0 ┤──────────────→ V (V)
           0.3  0.5  0.7  0.9
        
        锗管    硅管
```

**特点**：
1. 存在**死区电压**（门坎电压）：硅管约0.5V，锗管约0.1V
2. 正常工作电压：硅管0.6~0.8V，锗管0.2~0.3V
3. 正向电流随电压近似指数增长

### 3.2.2 反向特性

**特点**：
1. 反向电压在一定范围内，反向电流很小（近似等于 $I_S$）
2. 当 $|V_R| > |V_{BR}|$ 时，反向电流急剧增加，发生击穿
3. 硅管的反向电流比锗管小得多

### 3.2.3 温度特性

温度对二极管特性有显著影响：

| 参数 | 温度系数 | 说明 |
|------|----------|------|
| $V_F$（正向压降） | -2 mV/°C | 温度升高，$V_F$ 减小 |
| $I_S$（反向饱和电流） | 约翻倍/10°C | 温度升高，$I_S$ 急剧增大 |
| $V_{BR}$（击穿电压） | 取决于击穿类型 | 雪崩击穿正温度系数，齐纳击穿负温度系数 |

## 3.3 二极管的主要参数

| 参数 | 符号 | 说明 |
|------|------|------|
| 最大整流电流 | $I_F$（或 $I_{F(AV)}$） | 长期允许通过的最大正向平均电流 |
| 最高反向工作电压 | $V_R$（或 $V_{RM}$） | 允许承受的最大反向电压（通常取击穿电压的1/2~2/3） |
| 反向电流 | $I_R$ | 在最高反向工作电压下的反向电流 |
| 最高工作频率 | $f_M$ | 超过此频率，二极管失去单向导电性 |
| 正向压降 | $V_F$ | 正向电流为额定值时的正向电压降 |
| 动态电阻 | $r_d$ | 工作点处的交流电阻，$r_d = \frac{V_T}{I_D}$ |
| 结电容 | $C_j$ | PN结电容 |

## 3.4 二极管电路的分析方法

### 3.4.1 理想二极管模型

- 正偏：电压为0，相当于短路
- 反偏：电流为0，相当于开路

```
等效电路：

正偏：    A ────── K
    
反偏：    A ──┬──── K
            │ X │
            └───┘
```

### 3.4.2 恒压降模型

考虑二极管的实际正向压降：

$$V_D \approx \begin{cases} 0.7V & \text{（硅管）} \\ 0.3V & \text{（锗管）} \end{cases}$$

### 3.4.3 折线模型

将二极管特性曲线折线化：

$$I_D = \begin{cases} \frac{V_D - V_{on}}{r_D} & V_D > V_{on} \\ 0 & V_D \leq V_{on} \end{cases}$$

其中 $V_{on}$ 是开启电压，$r_D$ 是线性区的等效电阻。

### 3.4.4 小信号模型（交流模型）

在工作点 $Q$ 附近的小信号分析中：

$$r_d = \frac{\Delta v_D}{\Delta i_D} = \frac{V_T}{I_DQ}$$

其中 $I_{DQ}$ 是工作点电流。

## 3.5 特殊二极管

### 3.5.1 稳压二极管

**符号**：

```
    A ──▶┃◁── K
```

**特性**：
- 利用PN结的反向击穿特性
- 稳压管工作在反向击穿区
- 电流在很大范围内变化时，端电压基本不变

**稳压电路**：

```
        R
Vi ○──┤├──┬──○── Vo
            │  │
            │  ├──○── R_L
            │  │
            └──┤├─│←─┘
               ││
              D_Z
               │
               ── GND
```

**主要参数**：
- 稳定电压 $V_Z$：击穿后两端电压
- 稳定电流 $I_Z$：工作电流
- 动态电阻 $r_z = \frac{\Delta V_Z}{\Delta I_Z}$：越小越好
- 功耗 $P_Z = V_Z \cdot I_Z$

### 3.5.2 变容二极管

利用PN结电容随电压变化的特性，可作为压控电容器。

$$C_j = \frac{C_0}{(1 + V/V_{bi})^n}$$

其中 $n \approx 1/2$ ~ 1/3。

### 3.5.3 发光二极管(LED)

**原理**：某些半导体材料（如GaAs、GaP）在正向偏置下，电子与空穴复合时会发出光子。

**颜色与材料**：
| 材料 | 颜色 |
|------|------|
| GaAs | 红外 |
| GaAsP | 红、黄 |
| GaP | 绿 |
| SiC | 蓝 |

**特点**：
- 正向压降：1.5~3V
- 需要串联限流电阻

### 3.5.4 光电二极管

**原理**：反向偏置下，光照产生电子-空穴对，增加反向电流。

**特点**：
- 无光照时，反向电流很小（暗电流）
- 有光照时，反向电流随光强增加（光电流）

### 3.5.5 肖特基二极管

**原理**：金属与N型半导体接触形成肖特基势垒，具有单向导电性。

**特点**：
- 正向压降低（0.3~0.5V）
- 开关速度快
- 反向恢复时间短

**应用**：高频整流、电源开关、检波

## 3.6 二极管的应用电路

### 3.6.1 限幅电路

利用二极管的单向导电性，将输出电压限制在某一范围内。

```
双向限幅电路：

    R
vi○──┤├──┬──○── vo
         │  │
         │  ├──┤├──┤
         │        │  D1、D2串联
         │  ├──┤├──┤
         │  │  │
         └──┴──┴──┘
            │  │
           D1  D2
```

**输出特性**：
$$v_o = \begin{cases} V_{DD} + V_D & v_i > V_{DD} + V_D \\ v_i & -V_{DD} - V_D < v_i < V_{DD} + V_D \\ -(V_{DD} + V_D) & v_i < -V_{DD} - V_D \end{cases}$$

### 3.6.2 钳位电路

将电路中某点的电压钳制在某一固定值。

### 3.6.3 二极管保护电路

在感性负载两端反并联二极管，用于吸收线圈断电时产生的反向电动势：

```
感性负载保护：

    L ──┬──┬── Vo
        │  │
        └──┤├─▶┃──┘
           │
```

# 第4章 双极型晶体管(BJT)

## 4.1 BJT的结构与分类

### 4.1.1 基本结构

双极型晶体管（Bipolar Junction Transistor, BJT）由两个PN结、三个区域、三个电极组成。

```
NPN型三极管结构示意图：

       C(集电极)
        │
      ┌─┴─┐
      │ N │  集电区（面积大）
      └─┬─┘
        │P   基区（薄，几十μm）
      ┌─┴─┐
      │ N │  发射区（掺杂浓度高）
      └─┬─┘
        │
        E(发射极)
```

**三个区域**：
- **发射区(Emitter)**：发射载流子，掺杂浓度高
- **基区(Base)**：传输和控制载流子，宽度很薄，掺杂浓度低
- **集电区(Collector)**：收集载流子，面积大

**两个PN结**：
- **发射结(J_E)**：发射区与基区之间的PN结
- **集电结(J_C)**：集电区与基区之间的PN结

### 4.1.2 BJT的分类

| 按结构 | NPN型 | PNP型 |
|--------|-------|-------|
| 电路符号 | ```B─○─C```<br/>```B─○─E``` | ```B─○─C```<br/>```B─○─E``` |
| 载流子 | 电子是多子 | 空穴是多子 |
| 偏置要求 | E极电位最低 | E极电位最高 |

## 4.2 BJT的工作原理

### 4.2.1 放大工作的外部条件

要使BJT正常工作于放大状态，必须满足：

| 结型 | 偏置要求 |
|------|----------|
| 发射结 | 正向偏置（$V_{BE} \approx 0.6\sim0.7V$） |
| 集电结 | 反向偏置（$V_{CB} > 0$） |

**以NPN管为例**：

```
        VCC
         │
         R_C
         │
    C ───┼───○── Vo
         │   │
         │   └──▶┃ (C)
         │       │
        [B]──R_B─┤ (B)
         │       │    NPN
        [E]──────┘ (E)
         │
        VEE(通常接地)
```

### 4.2.2 载流子的传输过程

**以NPN管为例，载流子传输过程如下**：

**第一步：发射区向基区注入电子**

发射结正偏，发射区的多子（电子）向基区扩散注入，形成**发射极电流** $I_E$。

```
    发射区 → 电子 →→→→→→→→ 基区
    (N)    →      ↓↓↓
              空穴 ←←← 基区向发射区注入（很小）
```

**第二步：电子在基区的扩散与复合**

注入基区的电子成为非平衡少子，在浓度梯度作用下向集电结扩散。由于基区很薄且空穴浓度低，复合很少。

部分电子与基区的空穴复合，形成**基极电流** $I_B$ 的主要部分。

```
    基区中的电子分布：
    
    浓度
      ↑        ╭──╮
      │        ╱    ╲
      │       ╱      ╲
      │      ╱        ╲
      │     ╱          ╲
      │    ╱            ╲
      └───┴──────────────┴──→ x
           0    x_B
           
      x_B：基区宽度
      浓度梯度驱动电子向集电结扩散
```

**第三步：集电区收集电子**

集电结反偏，漂移作用强。扩散到集电结边缘的电子被电场扫入集电区，形成**集电极电流** $I_C$。

同时，集电区的少子（空穴）向基区漂移，形成反向饱和电流 $I_{CBO}$。

```
    电子传输：
    
    发射区 →→→→→ 基区 →→→→→ 集电区
      ↓         ↓           ↓
     扩散      复合        漂移
     少量       (少数)      全部
```

### 4.2.3 电流分配关系

定义以下电流：

| 电流 | 定义 |
|------|------|
| $I_E$ | 发射极电流 |
| $I_B$ | 基极电流 |
| $I_C$ | 集电极电流 |

**电流分配关系**：

$$I_E = I_C + I_B$$

$$I_C = \bar{\alpha} I_E + I_{CBO}$$

其中 $\bar{\alpha}$ 是共基极直流电流放大系数，通常接近于1（0.95~0.99）。

**共射极直流电流放大系数** $\bar{\beta}$：

$$\bar{\beta} = \frac{I_C}{I_B}$$

$$\bar{\alpha} = \frac{\bar{\beta}}{1+\bar{\beta}}$$

**三者之间的关系**：

$$I_E = (1+\bar{\beta})I_B$$
$$I_C = \bar{\beta}I_B$$
$$I_{CBO} = (1+\bar{\beta})I_{CBO}'$$

其中 $I_{CBO}'$ 是集电极-基极间的反向饱和电流。

## 4.3 BJT的特性曲线

### 4.3.1 输入特性曲线

$$I_B = f(V_{BE}) \big|_{V_{CE}=const}$$

```
        I_B (μA)
        ↑
    60 ┤      V_CE = 10V
        │     /
    40 ┤    /  V_CE = 5V
        │   /
    20 ┤  /  V_CE = 2V
        │ /
      0 ┼──────────────────→ V_BE (V)
        0.5  0.6  0.7  0.8
```

**特点**：
1. 形状与PN结正向特性相似
2. $V_{CE}$ 增大，曲线右移（基区宽度调制效应）
3. $V_{CE} \geq 1V$ 后，曲线基本重合

### 4.3.2 输出特性曲线

$$I_C = f(V_{CE}) \big|_{I_B=const}$$

```
        I_C (mA)
        ↑
    60 ┤  I_B=60μA  ─────────────
    50 ┤  I_B=50μA  ─────────────
    40 ┤  I_B=40μA  ─────────────
    30 ┤  I_B=30μA  ─────────────
    20 ┤  I_B=20μA  ─────────────
    10 ┤  I_B=10μA  ─────────────
      0 ┼──────────────────────→ V_CE (V)
        0              10        20
        ↑       ↑       ↑       ↑
      饱和区  放大区    放大区   击穿区
```

### 4.3.3 三个工作区域

| 区域 | 发射结 | 集电结 | $I_C$ 与 $I_B$ 的关系 | 特点 |
|------|--------|--------|----------------------|------|
| 放大区 | 正偏 | 反偏 | $I_C = \beta I_B$ | 具有电流放大作用 |
| 饱和区 | 正偏 | 正偏（0.3~0.7V） | $I_C < \beta I_B$ | $V_{CE} \approx 0.3V$ |
| 截止区 | 反偏 | 反偏 | $I_C \approx 0$ | $I_B \approx 0$ |

**饱和深度的定义**：

$$S_{sat} = \frac{I_{CS}}{I_C} = \frac{V_{CC}/R_C}{I_C}$$

其中 $I_{CS}$ 是集电极饱和电流。

## 4.4 BJT的主要参数

### 4.4.1 电流放大系数

| 参数 | 符号 | 典型值 | 说明 |
|------|------|--------|------|
| 共射直流电流放大系数 | $\bar{\beta}$ | 20~200 | $I_C/I_B$ |
| 共射交流电流放大系数 | $\beta$ | ≈ $\bar{\beta}$ | $\Delta I_C/\Delta I_B$ |
| 共基直流电流放大系数 | $\bar{\alpha}$ | 0.95~0.99 | $I_C/I_E$ |
| 共基交流电流放大系数 | $\alpha$ | ≈ $\bar{\alpha}$ | $\Delta I_C/\Delta I_E$ |

**$\beta$ 与 $\alpha$ 的关系**：

$$\alpha = \frac{\beta}{1+\beta}$$
$$\beta = \frac{\alpha}{1-\alpha}$$

### 4.4.2 极间反向电流

| 参数 | 符号 | 说明 |
|------|------|------|
| 集电极-基极反向饱和电流 | $I_{CBO}$ | 发射极开路时，C-B间的反向电流 |
| 穿透电流 | $I_{CEO}$ | 基极开路时，C-E间的电流，$I_{CEO} = (1+\beta)I_{CBO}$ |

**温度关系**：温度每升高10°C，$I_{CBO}$ 约增加一倍。

### 4.4.3 极限参数

| 参数 | 符号 | 说明 |
|------|------|------|
| 集电极最大允许电流 | $I_{CM}$ | 超过此值，$\beta$ 将明显下降 |
| 集电极最大允许功耗 | $P_{CM}$ | $P_C = V_{CE} \cdot I_C$ |
| 反向击穿电压 | $V_{(BR)CEO}$ | 基极开路时，C-E间的击穿电压 |

**安全工作区**：

```
        I_C (mA)
        ↑
   I_CM ┤─────── ___________
        │      /|         |
        │     / |         |
        │    /  |         |
        │   /   |         |
        │  /    |         |
        │ /     |         |
        │/______|_________|___ V_CE
              V_(BR)CEO
              
        ★ 安全工作区（SOA）
```

## 4.5 温度对BJT的影响

### 4.5.1 对 $V_{BE}$ 的影响

温度每升高1°C，$V_{BE}$ 约减小2mV：

$$\frac{dV_{BE}}{dT} \approx -2 \text{ mV/°C}$$

**影响**：在 $I_B$ 一定时，温度升高，$V_{BE}$ 减小。

### 4.5.2 对 $\beta$ 的影响

温度升高，$\beta$ 增大：

$$\frac{d\beta}{dT} \approx +1\%/\text{°C}$$

即温度每升高1°C，$\beta$ 约增加1%。

### 4.5.3 对 $I_{CBO}$ 的影响

温度每升高10°C，$I_{CBO}$ 约增加一倍。

### 4.5.4 温度对输出特性的影响

```
        I_C
         ↑
         │     温度升高
         │    →
         │   /
         │  /  虚线：高温
         │ /   
        /───────→ V_CE
       /
      /
     /
实线：低温
```

# 第5章 场效应管(FET)

## 5.1 场效应管概述

### 5.1.1 场效应管的特点

场效应管（Field Effect Transistor, FET）是一种**电压控制电流**的半导体器件，与BJT的电流控制特性不同。

**FET与BJT的比较**：

| 特性 | FET | BJT |
|------|-----|-----|
| 控制方式 | 电压控制 | 电流控制 |
| 输入电阻 | 极高（>10¹⁴Ω） | 较低（kΩ级） |
| 噪声 | 较小 | 较大 |
| 温度稳定性 | 好 | 较差 |
| 功耗 | 较低 | 较高 |
| 集成度 | 高 | 较低 |

### 5.1.2 FET的分类

```
场效应管分类：

          场效应管
             │
    ┌────────┴────────┐
    │                 │
  结型              绝缘栅型
  (JFET)            (MOSFET)
    │                 │
 ┌──┴──┐          ┌──┴──┐
 │     │          │     │
N沟道  P沟道      N沟道  P沟道
    │                 │
 ┌──┴──┐          ┌──┴──┐
 │增强型│          │增强型│
 │耗尽型│          │耗尽型│
 └─────┘          └─────┘
```

## 5.2 结型场效应管(JFET)

### 5.2.1 N沟道JFET的结构

```
JFET结构示意图（N沟道）：

         G(栅极)
          │
      ┌───┴───┐
      │  P区   │
      │(P区)   │
    ┌─┴────────┴─┐
    │   N区      │
    │  源极S     │
    │   N沟道    │
    └────────────┘
        D(漏极)
```

**结构特点**：
- 沟道位于两个P区之间
- 栅极与沟道之间形成PN结

### 5.2.2 JFET的工作原理

**当 $V_{GS} = 0$ 时**：

在漏-源电压 $V_{DS}$ 作用下，电子从源极流向漏极，形成漏极电流 $I_D$。

沟道类似于一个电阻，$I_D$ 随 $V_{DS}$ 近似线性增长（未夹断前）。

**当 $V_{GS} < 0$ 时（负偏）**：

栅极与沟道之间的PN结反偏，耗尽层变宽，沟道变窄。

```
不同 $V_{GS}$ 下的沟道形状：

V_GS = 0:     V_GS = V_P/2:    V_GS = V_P:
═════════    ════════      ══
████████    ████████      ██████████
████████    ██████████    ██████████
████████    ██████████    ██████████
═════════    ════════      ══
     宽沟道       窄沟道        夹断
```

**夹断电压 $V_P$**：

当 $|V_{GS}|$ 增大到某一值时，两个耗尽层在沟道中间相遇，沟道完全夹断，此时的栅源电压称为**夹断电压** $V_P$。

**转移特性**：

在饱和区（$V_{DS} > V_{GS} - V_P$）：

$$I_D = I_{DSS}\left(1 - \frac{V_{GS}}{V_P}\right)^2 \quad (V_P < V_{GS} < 0)$$

其中 $I_{DSS}$ 是 $V_{GS} = 0$ 时的漏极电流（饱和漏极电流）。

**输出特性**：

```
        I_D
         ↑
         │
    I_DSS ┤╲              恒流区（饱和区）
         │  ╲
         │   ╲ V_GS = 0
         │    ╲
         │     ╲ V_GS = V_P/2
         │      ╲
         │       ╲
         │        ╲ V_GS = V_P
         │         ╲___
         └──────────────────→ V_DS
              V_DS(sat)
```

## 5.3 绝缘栅场效应管(MOSFET)

### 5.3.1 N沟道增强型MOSFET

```
MOSFET结构示意图（N沟道增强型）：

         G(栅极)
          │
      ╔═══╧═══╗
      ║ SiO₂  ║  ← 绝缘层
      ╠═══╤═══╣
      │ P区    │
    ┌─┴───────┴─┐
    │   N区     │
    │  源极S    │
    └───────────┘
        D(漏极)
```

**特点**：
- 栅极与半导体之间有绝缘层（SiO₂）
- 输入电阻极高（>10¹⁴Ω）
- 栅极电流几乎为零
- $V_{GS} = 0$ 时没有导电沟道

### 5.3.2 N沟道增强型MOSFET的工作原理

**当 $V_{GS} < V_{TH}$ 时**：

栅极电压不够高，不能在P型衬底表面形成反型层，没有导电沟道，$I_D = 0$。

**当 $V_{GS} > V_{TH}$ 时**：

栅极电压在P型衬底表面感应出负电荷（电子），当 $V_{GS}$ 足够高时，表面形成N型反型层，成为连接源区和漏区的导电沟道。

**阈值电压 $V_{TH}$**：
- 典型值：0.5~3V
- 与栅氧化层厚度、衬底掺杂浓度等因素有关

**转移特性**：

$$I_D = K_n(V_{GS} - V_{TH})^2 \quad (V_{GS} > V_{TH})$$

其中 $K_n = \frac{\mu_n C_{ox} W}{2L}$ 是导电因子。

**输出特性**：

```
        I_D
         ↑
         │
         │  V_GS = 6V
         │╲
         │ ╲  V_GS = 5V
         │  ╲
         │   ╲ V_GS = 4V
         │    ╲
         │     ╲ V_GS = V_TH
         │      ╲
         │       ╲
         └────────╲────────→ V_DS
                    ╲___
                      V_DS(sat)
```

### 5.3.3 N沟道耗尽型MOSFET

**特点**：
- 制造时在栅氧化层下已存在原始沟道
- $V_{GS} = 0$ 时，已有漏极电流 $I_{DSS}$
- $V_{GS}$ 可正可负
- 负 $V_{GS}$ 减小沟道，正 $V_{GS}$ 增大沟道

**转移特性**：

$$I_D = I_{DSS}\left(1 - \frac{V_{GS}}{V_P}\right)^2$$

其中 $V_P$ 是夹断电压（与增强型中的 $V_{TH}$ 类似，但为负值）。

### 5.3.4 P沟道MOSFET

P沟道MOSFET与N沟道MOSFET类似，但：
- 衬底是N型
- 沟道是P型
- 载流子是空穴
- 迁移率约为N沟道的1/2
- $V_{GS}$ 和 $V_{TH}$ 为负值

## 5.4 FET的主要参数

| 参数 | 符号 | 说明 |
|------|------|------|
| 饱和漏极电流 | $I_{DSS}$ | $V_{GS}=0$ 时的漏极电流（耗尽型） |
| 夹断电压 | $V_P$ | JFET或耗尽型MOSFET的夹断电压 |
| 阈值电压 | $V_{TH}$ | 增强型MOSFET开始导通的栅压 |
| 跨导 | $g_m$ | $g_m = \frac{\partial I_D}{\partial V_{GS}}$，衡量栅压对漏流的控制能力 |
| 漏极电阻 | $r_{DS}$ | 输出电阻，$r_{DS} = \frac{\partial V_{DS}}{\partial I_D}$ |
| 最大漏极电流 | $I_{DM}$ | 最大允许漏极电流 |
| 最大功耗 | $P_{DM}$ | 最大允许功耗 |

**跨导公式**（饱和区）：

$$g_m = \frac{\partial I_D}{\partial V_{GS}} = 2K_n(V_{GS} - V_{TH})$$

或

$$g_m = \frac{2I_D}{V_{GS} - V_{TH}}$$

## 5.5 FET与BJT的比较

| 特性 | JFET | MOSFET | BJT |
|------|------|--------|-----|
| 输入电阻 | 高（10⁸Ω） | 极高（10¹⁴Ω） | 低（kΩ） |
| 栅极电流 | ≈0 | ≈0 | 有基极电流 |
| 噪声 | 低 | 低 | 较高 |
| 集成度 | 高 | 最高 | 较低 |
| 功耗 | 低 | 最低 | 较高 |
| 速度 | 较高 | 高 | 较低 |
| 控制方式 | 电压 | 电压 | 电流 |
| gm与ID关系 | $g_m \propto \sqrt{I_D}$ | $g_m \propto \sqrt{I_D}$ | $g_m \propto I_C$ |

# 第二篇 基本放大电路

# 第6章 基本放大电路的工作原理

## 6.1 放大的基本概念

### 6.1.1 放大的本质

"放大"不是凭空产生能量，而是用**小的输入信号控制大的输出信号**。放大电路实质上是一个**能量转换器**：将直流电源的能量转换为随输入信号变化的交流能量。

```
放大电路框图：

        直流电源 VCC
            │
            ↓
    ┌───────────────┐
    │               │
    │    放大器      │  ← 输出功率 > 输入功率
    │               │
 vi─┤               ├── vo
    │               │
    └───────────────┘
     (小信号)        (大信号)
```

### 6.1.2 放大的条件

1. **有源器件工作在放大区**：BJT的发射结正偏、集电结反偏
2. **建立合适的静态工作点**：使有源器件有适当的直流偏置
3. **信号能够正常传输**：输入信号能加到放大器件上，输出信号能送到负载

### 6.1.3 放大的性能指标

| 指标 | 符号 | 定义 |
|------|------|------|
| 电压放大倍数 | $A_v$ | $A_v = \frac{v_o}{v_i}$ |
| 电流放大倍数 | $A_i$ | $A_i = \frac{i_o}{i_i}$ |
| 功率放大倍数 | $A_p$ | $A_p = A_v \cdot A_i$ |
| 输入电阻 | $r_i$ | $r_i = \frac{v_i}{i_i}$ |
| 输出电阻 | $r_o$ | $r_o = \frac{v_o}{i_o}\big|_{R_L=\infty}$ |

## 6.2 基本共射放大电路

### 6.2.1 电路组成

```
基本共射放大电路：

        VCC
         │
         R_C
         │
    C₁ ──┼───○── Vo
         │   │
         │   └──▶┃ (C)
         │       │
        [B]──R_B─┤ (B)
         │       │     NPN
        [E]──────┘ (E)
         │
         ── GND
        
Vi ──C₁──┬───○───○── Vo
              B   C
```

**各元件作用**：

| 元件 | 作用 |
|------|------|
| $V_{CC}$ | 提供直流能量（集电极电源） |
| $R_C$ | 将集电极电流变化转变为电压变化 |
| $R_B$ | 为基极提供偏置电流（基极偏置电阻） |
| $C_1, C_2$ | 耦合电容，隔直通交 |
| 三极管T | 放大器件 |

### 6.2.2 工作原理

**直流分量（静态）**：
- $V_{CC}$ 通过 $R_B$ 为发射结提供正向偏置
- $V_{BE} \approx 0.7V$
- $I_B = (V_{CC} - V_{BE})/R_B$
- $I_C = \beta I_B$
- $V_{CE} = V_{CC} - I_CR_C$

**交流分量（动态）**：
- 输入信号 $v_i$ 通过 $C_1$ 加到基极
- $v_{BE}$ 变化引起 $i_B$ 变化
- $i_B$ 变化引起 $i_C$ 变化（$i_C = \beta i_B$）
- $i_C$ 变化在 $R_C$ 上产生电压变化
- 通过 $C_2$ 输出放大后的信号

**相位关系**：共射放大电路的输出与输入**相位相反**（相差180°）。

## 6.3 直流通路与交流通路

### 6.3.1 直流通路

**画法原则**：
- 电容视为**开路**
- 电感视为**短路**
- 信号源视为短路（但保留内阻）

**直流通路用于**：分析静态工作点

```
直流通路：

        VCC
         │
         R_C
         │
        [B]──R_B─┤ (B)
         │       │
        [E]──────┘ (E)
```

### 6.3.2 交流通路

**画法原则**：
- 大容量电容（耦合电容、旁路电容）视为**短路**
- 直流电源视为**短路**

**交流通路用于**：分析动态参数

```
交流通路：

        r_be
 vi ○──┤├├├┤├├├──┤├──○── vo
             │      │
             │     βi_b
             │      │
             └──────┘
```

## 6.4 静态工作点

### 6.4.1 静态工作点的定义

**静态工作点（Q点）**：无输入信号时，晶体管各极直流电压和电流。

$$Q(I_{BQ}, I_{CQ}, V_{CEQ})$$

### 6.4.2 静态工作点的计算

$$I_{BQ} = \frac{V_{CC} - V_{BE}}{R_B} \approx \frac{V_{CC}}{R_B}$$

$$I_{CQ} = \beta I_{BQ}$$

$$V_{CEQ} = V_{CC} - I_{CQ}R_C$$

**估算假设**：
- $V_{BE} \approx 0.7V$（硅管）
- $I_B \gg$ 流入基区的其他电流

### 6.4.3 静态工作点的图解分析

**输入回路的图解**：

$$\frac{V_{CC} - V_{BE}}{R_B} = I_B$$

在输入特性曲线上找到满足此方程的点。

**输出回路的图解**：

直流负载线方程：
$$V_{CE} = V_{CC} - I_CR_C$$

直流负载线与 $I_B = I_{BQ}$ 曲线的交点即为Q点。

```
        I_C
         ↑
    Q    ●  直流负载线
     ↗  / 
    /  /  
   /  /   
  /  /    
 /  /     
/  /      
───────────────→ V_CE
      Q     V_CEQ
```

## 6.5 动态分析

### 6.5.1 图解法分析动态

**交流负载线**：
- 通过Q点
- 斜率为 $-1/R_L'$，其中 $R_L' = R_C \parallel R_L$

$$v_{CE} = V_{CEQ} - i_c R_L'$$

**波形分析**：

```
输入信号较小、工作点合适时的波形：

v_BE  ───┐    ┌─────
      ───┘    └─────

i_B  ───┐    ┌─────
      ───┘    └─────

i_C  ───┴────┘──────
      ↑      ↑      ↑
     i_c    i_c    i_c
     
v_CE ───┬────┐───────
      ──┘    └──────
      ↑      ↑      ↑
     v_ce   v_ce   v_ce
```

### 6.5.2 电压放大倍数的图解

$$A_v = \frac{v_o}{v_i} = \frac{-i_c R_L'}{v_{be}} = -\frac{\beta i_b R_L'}{i_b r_{be}} = -\frac{\beta R_L'}{r_{be}}$$

负号表示输出与输入**相位相反**。

# 第7章 放大电路的分析方法

## 7.1 等效电路法

### 7.1.1 BJT的h参数等效电路

BJT的输入和输出特性可用双口网络描述：

**h参数定义**：

| 参数 | 定义 | 物理意义 |
|------|------|----------|
| $h_{ie}$ | $\left. \frac{\partial v_{BE}}{\partial i_B} \right|_{V_{CE}}$ | 输入电阻 |
| $h_{re}$ | $\left. \frac{\partial v_{BE}}{\partial v_{CE}} \right|_{I_B}$ | 反向电压传输系数 |
| $h_{fe}$ | $\left. \frac{\partial i_C}{\partial i_B} \right|_{V_{CE}}$ | 电流放大系数 |
| $h_{oe}$ | $\left. \frac{\partial i_C}{\partial v_{CE}} \right|_{I_B}$ | 输出电导 |

**h参数等效电路**：

```
         b
          │
          ├─────── h_ie ──────┐
          │                    │
         h_re·v_ce           i_c
          │                    │
          └────────┬───────────┤
                   │           │
                  c◄──────────┘
                  /
                 /
                1/h_oe
               /
              ○ e
```

### 7.1.2 简化的等效电路

在低频小信号条件下，$h_{re}$ 和 $h_{oe}$ 通常可以忽略：

```
         b
          │
     v_be │ h_ie (=r_be)
          ├───────
          │       │
         r_be   βi_b
          │       │
          └───────┴───── c
                     │
                     └────────○ e
```

### 7.1.3 r_be的计算

$$r_{be} = r_{bb'} + (1+\beta)\frac{V_T}{I_E}$$

其中：
- $r_{bb'}$：基区体电阻（约200~300Ω）
- $V_T$：热电压（约26mV）
- $I_E$：发射极静态电流（mA）

**简化公式**：

$$r_{be} \approx 300 + (1+\beta)\frac{26(mV)}{I_E(mA)} \quad (\Omega)$$

## 7.2 共射放大电路的动态分析

### 7.2.1 电压放大倍数

$$A_v = \frac{v_o}{v_i} = \frac{-\beta i_b R_L'}{i_b r_{be}} = -\frac{\beta R_L'}{r_{be}}$$

其中 $R_L' = R_C \parallel R_L$

**负号意义**：输出电压与输入电压相位相反（反相180°）。

### 7.2.2 输入电阻

从输入端看进去的等效电阻：

$$r_i = R_B \parallel r_{be}$$

通常 $R_B \gg r_{be}$，所以 $r_i \approx r_{be}$

### 7.2.3 输出电阻

将输入短路、负载开路，从输出端看进去的等效电阻：

$$r_o = R_C$$

## 7.3 静态工作点的稳定

### 7.3.1 温度对工作点的影响

温度升高时：
- $\beta$ 增大
- $I_{CBO}$ 增大
- $V_{BE}$ 减小

导致 $I_C$ 增大，工作点上移，可能进入饱和区。

### 7.3.2 分压式电流负反馈偏置电路

```
分压式偏置电路：

        VCC
         │
         R_B1
         │
    C ───┼───○── Vo
         │   │
         │   └──▶┃
         │       │
        [B]──R_E─┤
         │   │   │
         │   └───┤
        R_B2    C_E
         │       │
         ────────┘
```

**稳定原理**：
- $I_1 \gg I_B$，则 $V_B \approx \frac{R_{B2}}{R_{B1}+R_{B2}}V_{CC}$（固定）
- $V_E = V_B - V_{BE}$ 固定
- $I_E = V_E/R_E$ 固定
- $I_C \approx I_E$ 固定

**静态分析**：

$$V_B = \frac{R_{B2}}{R_{B1}+R_{B2}}V_{CC}$$

$$V_E = V_B - V_{BE}$$

$$I_E = \frac{V_E}{R_E}$$

$$I_{CQ} \approx I_E$$

$$V_{CEQ} = V_{CC} - I_{CQ}(R_C+R_E)$$

### 7.3.3 稳定系数

衡量工作点稳定性的指标：

$$S = \frac{\partial I_C}{\partial I_{CBO}} \bigg|_{V_{BE}, \beta = const}$$

理想情况下 $S = 1$，实际约2~10。

## 7.4 失真分析

### 7.4.1 饱和失真

**原因**：Q点过高，$i_C$ 的正半周进入饱和区

**现象**：输出波形底部被削平

```
饱和失真波形：

     vo
      ↑
      │    底部被削平
      │
──────┼──────────────→ t
      │
      │
      ↓
```

### 7.4.2 截止失真

**原因**：Q点过低，$i_C$ 的负半周进入截止区

**现象**：输出波形顶部被削平

```
截止失真波形：

     vo
      ↑
      │
──────┼──────────────→ t
      │
      │    顶部被削平
      ↓
```

### 7.4.3 失真的消除

| 失真类型 | 消除方法 |
|----------|----------|
| 饱和失真 | 减小 $R_B$（增大 $I_B$），减小 $R_C$，增大 $R_E$ |
| 截止失真 | 增大 $R_B$（减小 $I_B$），减小 $R_E$ |

# 第8章 放大电路的三种组态

## 8.1 共射放大电路

### 8.1.1 电路图

```
        VCC
         │

    C ───┼───○── Vo
```

| $r_o$ | $R_C$，较大 |

### 8.1.3 应用

- 多级放大电路的中间级

## 8.2 共集放大电路（射极输出器）

### 8.2.1 电路图

```
共集放大电路：

        VCC
         │
         R_B
         │
        [B]──┤ (B)
         │   │

                 │
```

### 8.2.2 性能特点

**电压放大倍数**：

$$A_v = \frac{(1+\beta)R_L'}{r_{be}+(1+\beta)R_L'} \approx 1$$

其中 $R_L' = R_E \parallel R_L$

**特点**：
- $A_v \approx 1$（略小于1）
- 电压跟随（相位相同）
- 输入电阻高：$r_i = R_B \parallel [(1+\beta)R_L']$
- 输出电阻低：$r_o = R_E \parallel \frac{r_{be}}{1+\beta}$

| 指标 | 特点 |
|------|------|
| $A_v$ | ≈1 |
| $A_i$ | 较大（$(1+\beta)$倍） |
| $r_i$ | 大（几十kΩ以上） |
| $r_o$ | 小（几十Ω） |
| 相位 | 0°（同相） |

### 8.2.3 应用

- **阻抗变换**：高输入阻抗，低输出阻抗
- **缓冲级**：隔离前后级影响
- **功率放大前置级**：推动功放电路

|------|------|

| $A_i$ | ≈1 |

| 高频特性 | 好（频率响应宽） |

### 8.3.3 应用

# 第9章 差分放大电路

## 9.1 直接耦合放大电路的问题

### 9.1.1 零点漂移

**定义**：当放大电路的输入端短路时，输出端却存在缓慢变化的电压。

**产生原因**：
- 温度变化
- 电源电压波动
- 元器件参数变化

**危害**：漂移电压与有用信号混杂，无法区分，严重时可能淹没有效信号。

### 9.1.2 解决方案

1. 在输入级采用**差分放大电路**
2. 采用**调制型直流放大器**
3. 采用**热稳定性好的电路**

## 9.2 差分放大电路的工作原理

### 9.2.1 基本差分放大电路

```
基本差分放大电路：

    VCC                  VCC
     │                   │
     R_C                 R_C
     │                   │
  ┌──┴──┐             ┌──┴──┐
  │ T₁  │             │ T₂  │
  └──┬──┘             └──┬──┘
     │                     │
     ├────○── v_o1    v_o2 ○──┤
     │                     │
  ┌──┴──┐             ┌──┴──┐
  │ T₁  │             │ T₂  │
  └──┬──┘             └──┬──┘
     │                     │
    R_E                   R_E
     │                     │
     └─────────┬───────────┘
               │
              GND
              
vi1 ──○─────────┼──────────○── vi2
```

### 9.2.2 差模信号与共模信号

**差模信号**：$v_{id} = v_{i1} - v_{i2}$，两输入端的**差值**

**共模信号**：$v_{ic} = \frac{v_{i1} + v_{i2}}{2}$，两输入端的**平均值**

**任意输入信号可以分解为**：

$$v_{i1} = v_{ic} + \frac{v_{id}}{2}$$
$$v_{i2} = v_{ic} - \frac{v_{id}}{2}$$

## 9.3 差分放大电路的分析

### 9.3.1 对差模信号的放大

**差模电压放大倍数**：

$$A_{vd} = \frac{v_{od}}{v_{id}} = \frac{v_{o1} - v_{o2}}{v_{i1} - v_{i2}}$$

**单端输出时**：

$$A_{vd1} = \frac{v_{o1}}{v_{id}} = \frac{1}{2}\frac{\beta R_C}{r_{be}}$$

**双端输出时**：

$$A_{vd} = \frac{v_{o1} - v_{o2}}{v_{i1} - v_{i2}} = \frac{\beta R_C}{r_{be}}$$

### 9.3.2 对共模信号的抑制

**共模电压放大倍数**：

理想情况下，$A_{vc} = 0$（完全抑制共模信号）

实际中，$A_{vc}$ 很小但不为零。

### 9.3.3 共模抑制比

**定义**：

$$K_{CMR} = \left| \frac{A_{vd}}{A_{vc}} \right|$$

或用分贝表示：

$$K_{CMR(dB)} = 20\lg\left| \frac{A_{vd}}{A_{vc}} \right| \text{ dB}$$

**意义**：$K_{CMR}$ 越大，差分放大电路抑制共模信号的能力越强。

**理想值**：$K_{CMR} = \infty$（即 $A_{vc} = 0$）

**实际值**：$K_{CMR} = 60$ ~ $120$ dB

## 9.4 改进型差分放大电路

### 9.4.1 射极耦合电阻R_E的作用

R_E对共模信号有负反馈作用：
- 共模信号使两边电流同时增大，$V_E$ 升高，抑制共模信号
- 差模信号使两边电流一增一减，$V_E$ 基本不变，不影响差模放大

**共模抑制比与R_E的关系**：

$$K_{CMR} \propto \frac{R_E}{r_e}$$

增大R_E可以提高共模抑制比。

### 9.4.2 恒流源偏置

用恒流源替代R_E，可以在不增大直流压降的情况下获得很大的等效电阻：

$$r_o \approx \frac{\beta V_T}{I_E} \approx \frac{26mV}{I_E(mA)} \times \beta \approx \text{几十kΩ到几MΩ}$$

```
恒流源偏置差分放大电路：

        VCC
         │
         R_C
         │
      ┌──┴──┐
      │ T₁  │
      └──┬──┘
         │
      ┌──┴──┐
      │ T₃  │ ← 恒流源
      └──┬──┘
         │I₀
      ┌──┴──┐
      │ T₂  │
      └──┬──┘
         │
        R_E
         │
        GND
```

## 9.5 差分放大电路的输入输出方式

| 方式 | 输入 | 输出 |
|------|------|------|
| 双端输入-双端输出 | 两端 | 两端 |
| 双端输入-单端输出 | 两端 | 一端 |
| 单端输入-双端输出 | 一端 | 两端 |
| 单端输入-单端输出 | 一端 | 一端 |

# 第10章 功率放大电路

## 10.1 功率放大电路的特点

### 10.1.1 主要问题

1. **输出功率大**：要求输出电压和电流的幅度都很大
2. **效率高**：减少能量损耗，节约能源
3. **非线性失真小**：输出波形应与输入波形相似
4. **散热问题**：功率管的散热与保护

### 10.1.2 功率指标

**输出功率**：

$$P_o = I_o \cdot V_o = \frac{1}{2}I_{om} \cdot V_{om}$$

**电源提供的直流功率**：

$$P_V = V_{CC} \cdot I_C$$

**效率**：

$$\eta = \frac{P_o}{P_V} \times 100\%$$

### 10.1.3 功率管的安全工作

- $P_{CM}$：集电极最大允许功耗
- $I_{CM}$：集电极最大允许电流
- $V_{(BR)CEO}$：反向击穿电压

## 10.2 功率放大电路的工作状态

| 类型 | 导通角 | 效率 | 失真 |
|------|--------|------|------|
| 甲类 | 360° | ≤50% | 小 |
| 乙类 | 180° | ≤78.5% | 交越失真 |
| 甲乙类 | 180°~360° | 50%~78.5% | 较小 |
| 丙类 | <180° | >78.5% | 大 |

## 10.3 乙类互补对称功率放大电路

### 10.3.1 OCL电路（无输出电容）

```
OCL电路：

        VCC
         │
      ┌──┴──┐
      │ NPN │
      └──┬──┘
         │T1
         ├────○─── Vo ─── R_L
         │
      ┌──┴──┐
      │ PNP │
      └──┬──┘
         │T2
         │
         GND
```

**工作原理**：
- $v_i > 0$：T1导通，T2截止，输出正半周
- $v_i < 0$：T2导通，T1截止，输出负半周
- 两管轮流导通，互相补充

**最大输出功率**：

$$P_{om} = \frac{V_{CC}^2}{2R_L}$$

**效率**：

$$\eta = \frac{\pi}{4} \approx 78.5\%$$

### 10.3.2 交越失真

乙类放大器在两管交替点附近存在失真，称为**交越失真**。

**产生原因**：
- 晶体管存在死区电压（0.5V左右）
- 当 $|v_i| < V_{on}$ 时，两个管子都不导通

```
交越失真波形：

     vo
      ↑
      │   /
      │  /
      │ /  
──────┼───────────────→ t
     /│
    / │
   /  │
      ↑
    交越失真区域
```

### 10.3.3 甲乙类互补对称电路

为消除交越失真，设置较小的静态偏置，使T1、T2工作在甲乙类状态。

```
甲乙类OCL电路：

        VCC
         │
      ┌──┴──┐
      │ NPN │
      └──┬──┘
         ├────○─── Vo ─── R_L
         │
      ┌──┴──┐
      │ PNP │
      └──┬──┘
         │
         │T3
      R│
      └┬┴─○── V_BB
       │
```

**V_BB的作用**：提供较小的正向偏压，消除死区影响。

### 10.3.4 OTL电路（无变压器）

```
OTL电路：

        VCC/2
         │
        R_L
         │
      ┌──┴──┐
      │ NPN │
      └──┬──┘
         ├────○─── Vo ──┤C│── R_L
         │
      ┌──┴──┐
      │ PNP │
      └──┬──┘
         │
         GND
```

**特点**：
- 单电源供电
- 输出电容 $C$ 起"负电源"作用
- 输出电容容量需足够大：$C \gg \frac{1}{\omega R_L}$

# 第三篇 集成运算放大器

# 第11章 集成运算放大器

## 11.1 集成运放概述

### 11.1.1 集成运放的特点

1. **高增益**：开环电压增益 $A_{vo}$ 可达 $10^5$ 以上
2. **高输入阻抗**：输入电阻 $r_i$ 可达 $10^6$Ω 以上
3. **低输出阻抗**：输出电阻 $r_o$ 约几百欧
4. **低漂移**：失调电压、失调电流温漂小
5. **通用性强**：应用广泛

### 11.1.2 电路符号

```
           ┌─────┐
    v+ ────┤     │
           │  ∇  ├─── v_o
    v- ────┤     │
           └─────┘
     +    -       out
    同相 反相
```

**基本关系**：
$$v_o = A_{vo}(v_+ - v_-)$$

## 11.2 集成运放的主要参数

| 参数 | 符号 | 典型值 | 说明 |
|------|------|--------|------|
| 开环增益 | $A_{vo}$ | 10⁵~10⁶ | 无反馈时的电压增益 |
| 输入失调电压 | $V_{IO}$ | mV级 | 使输出为零的输入差分电压 |
| 输入偏置电流 | $I_{IB}$ | nA~pA级 | 两输入端电流的平均值 |
| 输入失调电流 | $I_{IO}$ | nA~pA级 | 两输入端电流之差 |
| 输入电阻 | $r_i$ | MΩ~GΩ级 | 差模输入电阻 |
| 输出电阻 | $r_o$ | <100Ω |  |
| 共模抑制比 | $K_{CMR}$ | >100dB |  |
| 转换速率 | $S_R$ | V/μs级 | $S_R = \left|\frac{dv_o}{dt}\right|_{max}$ |
| 增益带宽积 | $GBW$ | MHz级 | $GBW = A_{vo} \cdot f_H$ |

## 11.3 理想运放特性

### 11.3.1 理想运放的两个重要特性

**1. 虚短**：

$$v_+ \approx v_-$$

两输入端电位近似相等。这是因为 $A_{vo}$ 非常大，$v_+ - v_- = v_o/A_{vo} \approx 0$。

**2. 虚断**：

$$i_+ \approx i_- \approx 0$$

两输入端电流近似为零。这是因为运放输入电阻非常高。

### 11.3.2 虚短与虚断的应用

利用虚短和虚断，可以方便地分析各种运放应用电路。

# 第12章 负反馈技术

## 12.1 反馈的基本概念

### 12.1.1 反馈的定义

将放大电路输出信号的一部分或全部，通过反馈网络送回输入端，与输入信号比较（加或减），这个过程称为**反馈**。

### 12.1.2 反馈的分类

**按反馈极性分**：

| 类型 | 定义 | 效果 |
|------|------|------|
| 正反馈 | 反馈信号增强净输入 | 增益提高，稳定性变差 |
| 负反馈 | 反馈信号削弱净输入 | 增益降低，稳定性提高 |

**按采样方式分**：

| 类型 | 采样对象 |
|------|----------|
| 电压反馈 | 输出电压 |
| 电流反馈 | 输出电流 |

**按比较方式分**：

| 类型 | 输入端连接方式 |
|------|----------------|
| 串联反馈 | 反馈信号与输入信号串联 |
| 并联反馈 | 反馈信号与输入信号并联 |

## 12.2 负反馈的四种组态

| 组态 | 采样 | 比较 | 输入电阻 | 输出电阻 |
|------|------|------|----------|----------|
| 电压串联 | 电压 | 串联 | ↑↑ | ↓↓ |
| 电压并联 | 电压 | 并联 | ↓ | ↓↓ |
| 电流串联 | 电流 | 串联 | ↑↑ | ↑↑ |
| 电流并联 | 电流 | 并联 | ↓ | ↑↑ |

## 12.3 负反馈对放大电路性能的影响

### 12.3.1 增益

闭环增益：
$$A_f = \frac{A}{1+AF}$$

其中 $1+AF$ 称为**反馈深度**。

当 $|1+AF| \gg 1$（深负反馈）时：
$$A_f \approx \frac{1}{F}$$

### 12.3.2 稳定性的改善

$$\frac{dA_f}{A_f} = \frac{1}{1+AF} \cdot \frac{dA}{A}$$

负反馈提高了增益的稳定性。

### 12.3.3 扩展通频带

$$f_{hf} = (1+AF)f_h$$
$$f_{lf} = \frac{f_l}{1+AF}$$

### 12.3.4 减小非线性失真

负反馈可以减小由放大器非线性引起的失真。

### 12.3.5 对输入/输出电阻的影响

| 反馈类型 | 输入电阻 | 输出电阻 |
|----------|----------|----------|
| 电压负反馈 | ↑ | ↓ |
| 电流负反馈 | ↑ | ↑ |
| 串联负反馈 | ↑ | - |
| 并联负反馈 | ↓ | - |

# 第13章 运算电路

## 13.1 比例运算电路

### 13.1.1 反相比例运算电路

```
反相比例运算电路：

        R_1
 vi ○──┤├──┬──○── vo
             │
           ┌─┴─┐
           │   │
           │ - │◄──反馈
           │   │
           └─┬─┘
             │
            R_F
             │
             ── GND
```

**分析**：
- $v_+ = 0$（接地）
- $v_- \approx v_+ = 0$（虚短）
- $i_1 = i_F$（虚断）
- $\frac{v_i - 0}{R_1} = \frac{0 - v_o}{R_F}$

**电压增益**：
$$A_{vf} = \frac{v_o}{v_i} = -\frac{R_F}{R_1}$$

**输入电阻**：
$$r_i = R_1$$

### 13.1.2 同相比例运算电路

```
同相比例运算电路：

 vi ○──┬───○── vo
        │   │
       R_1 │   │
        └───┤- │
            │   │◄──反馈
           +│   │
            └───┤
               R_F
                │
                ── GND
```

**电压增益**：
$$A_{vf} = 1 + \frac{R_F}{R_1}$$

**当 $R_F = 0, R_1 = \infty$ 时**：
$$A_{vf} = 1$$

成为**电压跟随器**。

## 13.2 加法运算电路

### 13.2.1 反相加法器

```
反相加法器：

 v1○──R_1─┐
          │
 v2○──R_2─┼──┬──○── vo
          │  │
 v3○──R_3─┘  │
              │
            ┌─┴─┐
            │ - │
            └─┬─┘
              │
             R_F
              │
              ── GND
```

**输出电压**：
$$v_o = -\left(\frac{R_F}{R_1}v_1 + \frac{R_F}{R_2}v_2 + \frac{R_F}{R_3}v_3\right)$$

**当 $R_1 = R_2 = R_3 = R$ 时**：
$$v_o = -\frac{R_F}{R}(v_1 + v_2 + v_3)$$

## 13.3 减法运算电路

```
减法电路：

 v1○──R_1─┬──┬──○── vo
          │  │
          │ ┌┴─┐
          │ │  │
          │ │ - │◄──反馈
          │ └┬─┘
          │  │
 v2○──R_2─┘  │
             R_F
              │
              ── GND
```

**输出电压**：
$$v_o = \frac{R_F}{R_1}v_1 - \frac{R_F}{R_2}v_2$$

**当 $R_1 = R_2 = R_F$ 时**：
$$v_o = v_1 - v_2$$

## 13.4 积分运算电路

```
积分电路：

 vi ○──R_1─┬──┬──○── vo
              │
            ┌─┴─┐
            │   │
            │ - │◄──反馈
            │   │ C_F
            └─┬─┼───┘
              │  │
              │  ── C_F
              │  │
              └──┘
```

**输出电压**：
$$v_o = -\frac{1}{R_1C_F}\int v_i \, dt$$

## 13.5 微分运算电路

```
微分电路：

 vi ○──C_1─┬──┬──○── vo
              │
            ┌─┴─┐
            │   │
            │ - │◄──反馈
            │   │ R_F
            └─┬─┤
              │  │
              │  R_F
              │  │
              └──┘
```

**输出电压**：
$$v_o = -R_F C_1 \frac{dv_i}{dt}$$

# 第14章 有源滤波器

## 14.1 滤波器的基本概念

### 14.1.1 滤波器的分类

| 类型 | 通带 | 阻带 |
|------|------|------|
| 低通滤波器(LPF) | 0 ~ $f_c$ | $f_c$ ~ ∞ |
| 高通滤波器(HPF) | $f_c$ ~ ∞ | 0 ~ $f_c$ |
| 带通滤波器(BPF) | $f_{L}$ ~ $f_{H}$ | 两端 |
| 带阻滤波器(BEF) | 两端 | $f_{L}$ ~ $f_{H}$ |

### 14.1.2 主要参数

- **通带增益**：通带内电压放大倍数
- **截止频率**：增益下降到通带增益的 $1/\sqrt{2}$ 处
- **过渡带宽**：通带到阻带的过渡宽度
- **品质因数Q**：$Q = \frac{f_0}{BW}$

## 14.2 一阶滤波器

### 14.2.1 一阶低通滤波器

```
    R ──┬──┬──○── vo
        │  │
        C  R_L
        │  │
       GND │
        └──┘
```

**传递函数**：
$$A(j\omega) = \frac{1}{1+j\omega RC}$$

**截止频率**：
$$f_c = \frac{1}{2\pi RC}$$

**幅频特性**：
$$|A| = \frac{1}{\sqrt{1+(\omega RC)^2}}$$

**衰减斜率**：-20dB/十倍频程

### 14.2.2 一阶高通滤波器

```
    C ──┬──┬──○── vo
        │  │
        R  R_L
        │  │
       GND │
        └──┘
```

**传递函数**：
$$A(j\omega) = \frac{j\omega RC}{1+j\omega RC}$$

**截止频率**：
$$f_c = \frac{1}{2\pi RC}$$

## 14.3 二阶滤波器

### 14.3.1 二阶低通滤波器

```
压控电压源(VCVS)二阶低通滤波器：

      R₁       C₁
 vi ○──┤├──┬──○──┤├──┬──○── vo
            │      │   │
            │      │   │
           R_F     │   │
            │      │   │
            └──────┴───┘
                 │
                C₂
                 │
                GND
```

**特点**：
- 滤波效果比一阶好
- -40dB/十倍频程下降斜率
- 可通过改变元件值调节 $f_c$

# 第四篇 信号产生与电源电路

# 第15章 正弦波振荡电路

## 15.1 振荡的条件

### 15.1.1 起振条件

$$\begin{cases} |AF| > 1 & \text{幅度条件} \\ \varphi_{AF} = 2n\pi & (n=0,1,2...) \end{cases}$$

### 15.1.2 平衡条件（稳幅振荡）

$$\begin{cases} |AF| = 1 & \text{幅值平衡} \\ \varphi_{AF} = 2n\pi & \text{相位平衡} \end{cases}$$

## 15.2 RC正弦波振荡器

### 15.2.1 RC桥式振荡器（文氏桥振荡器）

**选频网络**：RC串并联网络

**振荡频率**：
$$f_0 = \frac{1}{2\pi RC}$$

**起振条件**：
$$A_v = 1 + \frac{R_F}{R} > 3$$

## 15.3 LC正弦波振荡器

### 15.3.1 变压器反馈式振荡器

利用LC谐振回路选频，变压器耦合反馈。

**振荡频率**：
$$f_0 = \frac{1}{2\pi\sqrt{LC}}$$

### 15.3.2 电感三点式（哈特莱振荡器）

**振荡频率**：
$$f_0 = \frac{1}{2\pi\sqrt{(L_1+L_2)C}}$$

### 15.3.3 电容三点式（考毕兹振荡器）

**振荡频率**：
$$f_0 = \frac{1}{2\pi\sqrt{L\frac{C_1C_2}{C_1+C_2}}}$$

## 15.4 石英晶体振荡器

### 15.4.1 石英晶体的压电效应

- 压电效应：机械振动 ↔ 电振荡
- 谐振频率稳定，$Q$值极高（10⁴~10⁶）

### 15.4.2 等效电路与阻抗特性

- **串联谐振**：$f_s = \frac{1}{2\pi\sqrt{LC}}$，阻抗最小
- **并联谐振**：$f_p = \frac{1}{2\pi\sqrt{L\frac{CC_1}{C+C_1}}}$，阻抗最大

# 第16章 直流稳压电源

## 16.1 直流稳压电源的组成

```
交流输入 → 变压器 → 整流电路 → 滤波电路 → 稳压电路 → 直流输出
```

| 环节 | 功能 |
|------|------|
| 变压器 | 电压变换 |
| 整流 | 将交流变为脉动直流 |
| 滤波 | 减小输出纹波 |
| 稳压 | 稳定输出电压 |

## 16.2 整流电路

### 16.2.1 单相桥式整流

**输出电压平均值**：
$$V_{o(AV)} = 0.9V_2$$

**特点**：
- 变压器无需中心抽头
- 使用4个二极管
- 输出脉动频率是输入的2倍

## 16.3 滤波电路

### 16.3.1 电容滤波

**输出电压**：
$$V_{o(AV)} \approx \begin{cases} 1.2V_2 & \text{空载} \\ 1.0V_2 & \text{轻载} \\ 0.9V_2 & \text{重载} \end{cases}$$

### 16.3.2 电感滤波

适用于大电流负载。

### 16.3.3 π型滤波

电容-电感-电容（C-L-C）组合，滤波效果最好。

## 16.4 稳压电路

### 16.4.1 稳压管稳压电路

**稳压条件**：
$$I_Z_{min} < I_Z < I_Z_{max}$$

**限流电阻**：
$$R = \frac{V_i - V_Z}{I_Z + I_L}$$

### 16.4.2 串联型稳压电路

```
串联型稳压电路：

         R1
Vi ○──┤├──┬──○── Vo
           │  │
           │  ├──○── R_L
           │  │
          ┌┴──┴┐
          │ 调整管 │
          │   T   │
          └──┬──┘
             │
           ┌─┴─┐
        +──│比较│───│──○──
           │放大│
        -──└─┬─┘
             │
             │ Vref
           D_Z
```

**输出电压**：
$$V_o = V_Z \cdot \frac{R_1 + R_2}{R_2}$$

## 16.5 集成稳压器

### 16.5.1 三端集成稳压器

| 类型 | 型号 | 输出电压 |
|------|------|----------|
| 固定正压 | 78XX | +5V, +9V, +12V, +15V, +18V, +24V |
| 固定负压 | 79XX | -5V, -9V, -12V, -15V, -18V, -24V |
| 可调正压 | LM317 | 1.25V~37V |
| 可调负压 | LM337 | -1.25V~-37V |

# 第17章 电流源电路与基准源

## 17.1 电流源电路

### 17.1.1 镜像电流源

```
镜像电流源：

        VCC
         │
         R
         │
      ┌──┴──┐
      │ T₁  │
      └──┬──┘
         │I_R
      ┌──┴──┐
      │ T₂  │ ← I_O
      └──┬──┘
         │
```

**电流关系**：
$$I_O = I_R = \frac{V_{CC} - V_{BE}}{R}$$

### 17.1.2 微电流电流源

在镜像电流源中串联电阻，输出更小的电流。

$$I_O = \frac{V_T}{R_E}\ln\frac{I_R}{I_O}$$

### 17.1.3 电流源的应用

1. 替代大电阻作为有源负载
2. 为差分放大电路提供稳定的静态电流
3. 作为偏置电流源

## 17.2 基准电压源

### 17.2.1 带隙基准源

利用硅的带隙电压（约1.22V）作为参考：

$$V_{REF} = V_{BE} + K \cdot V_T$$

典型值：1.22V或2.5V

**特点**：
- 温度系数低
- 精度高
- 噪声低

# 第18章 模拟乘法器

## 18.1 模拟乘法器的基本原理

### 18.1.1 功能

模拟乘法器的输出是两个输入信号的乘积：

$$v_o = K \cdot v_x \cdot v_y$$

其中 $K$ 是乘法器的增益系数。

### 18.1.2 应用

| 应用领域 | 说明 |
|----------|------|
| 乘法运算 | $v_o = K \cdot v_x \cdot v_y$ |
| 乘方运算 | $v_o = K \cdot v_x^2$ |
| 除法运算 | $v_o = K \cdot \frac{v_x}{v_y}$ |
| 开方运算 | $v_o = K \cdot \sqrt{v_x}$ |
| 调制解调 | AM、DSB、FM等 |
| 功率测量 | $P \propto V^2$ |

## 18.2 乘法器电路

### 18.2.1 对数-指数乘法器

利用对数和指数运算实现乘法：

$$\ln v_o = \ln v_x + \ln v_y$$
$$v_o = e^{\ln v_x + \ln v_y} = v_x \cdot v_y$$

### 18.2.2 变跨导乘法器

利用BJT的指数特性：

$$I_C = I_S e^{V_{BE}/V_T}$$

双平衡乘法器（Gilbert单元）是集成电路中常用的乘法器结构。
