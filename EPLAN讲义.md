# 第1章 EPLAN概述

## 1.1 EPLAN简介

EPLAN Electric P8 是德国EPLAN公司开发的电气工程设计软件，主要用于电气控制系统的设计、文档管理和自动化。EPLAN是全球领先的电气CAD解决方案，被广泛应用于机械制造、汽车工业、电力系统、工业自动化等领域。

### 1.1.1 主要版本

| 版本 | 说明 |
|------|------|
| EPLAN Electric P8 | 基础版本 |
| EPLAN Electric P8 Professional | 专业版本，增加高级功能 |
| EPLAN Platform | 平台版本，整合多个EPLAN产品 |

### 1.1.2 主要应用领域

- 工业控制柜设计
- 自动化生产线电气设计
- 电力系统设计
- 设备电气维护文档
- 工程项目电气图纸

## 1.2 EPLAN优势

相比传统CAD软件，EPLAN具有以下优势：

| 特性 | 说明 |
|------|------|
| 数据库驱动 | 基于SQL数据库，设备信息统一管理 |
| 模块化设计 | 支持宏、模板，提高设计效率 |
| 自动编号 | 自动生成线号、器件编号 |
| 交叉引用 | 自动显示元件间的电气连接关系 |
| 报表生成 | 自动生成BOM表、接线表等 |
| 多语言支持 | 支持国际化项目 |
| 项目管理 | 完整的项目版本控制 |

## 1.3 EPLAN与AutoCAD的区别

| 对比项 | EPLAN | AutoCAD |
|--------|-------|---------|
| 定位 | 专业电气设计 | 通用绘图 |
| 数据管理 | 数据库驱动 | 文件驱动 |
| 符号库 | 专业电气符号 | 通用符号 |
| 自动编号 | 内置自动功能 | 需要手动 |
| 报表生成 | 自动生成多种报表 | 需要手动统计 |
| 电气功能 | 完整的电气设计功能 | 基础绘图 |

# 第3章 项目管理

## 3.1 项目概念

EPLAN中的项目是一个包含所有电气设计数据的容器，包括原理图、符号、设备信息、报表等。

### 3.1.1 项目文件结构

```
项目文件夹/
├── 项目名.edb/          # 数据库文件夹
│   ├── data/           # 项目数据
│   ├── index/          # 索引文件
│   └── ...
├── 项目名.elk           # 项目主文件
├── 项目名.prj/          # 项目配置
├── 生成/               # 报表输出目录
│   ├── PDF/
│   ├── BOM/
│   └── ...
└── 备份/               # 备份文件
```

### 3.1.2 项目类型

| 类型 | 说明 |
|------|------|
| 标准项目 | 常规电气设计项目 |
| 宏项目 | 用于创建和管理宏 |
| 部件库项目 | 用于管理部件库 |
| 升级项目 | 从旧版本升级的项目 |

## 3.2 创建新项目

### 3.2.1 新建项目步骤

**步骤1**：执行「项目」→「新建」

**步骤2**：在「新建项目」对话框中设置：

| 设置项 | 说明 | 示例 |
|--------|------|------|
| 名称 | 项目名称 | 电机控制系统 |
| 路径 | 保存位置 | D:\Projects\ |
| 模板 | 项目模板 | IEC_tpl |
| 结构 | 项目结构 | 功能+位置 |

**步骤3**：点击「确定」创建项目

### 3.2.2 项目模板

EPLAN提供多种预定义模板：

| 模板 | 适用场景 |
|------|----------|
| IEC_tpl | IEC标准模板 |
| NEMA_tpl | NEMA标准模板 |
| GB_tpl | 中国国家标准模板 |
| 空白模板 | 从头开始设计 |

### 3.2.3 项目结构

```
结构层级说明：

高层代号 (功能)
├── 系统A
│   ├── 子系统A1
│   └── 子系统A2
└── 系统B

位置代号 (安装位置)
├── 柜体1
│   ├── 安装板1
│   └── 安装板2
└── 柜体2
```

## 3.3 项目设置

### 3.3.1 项目属性

通过「项目」→「属性」可设置：

| 属性 | 说明 |
|------|------|
| 项目名称 | 项目标识名称 |
| 项目编号 | 项目唯一编号 |
| 设计者 | 项目设计人员 |
| 创建日期 | 项目创建时间 |
| 版本 | 项目版本号 |
| 客户 | 客户名称 |

### 3.3.2 图形编辑设置

| 设置项 | 说明 |
|--------|------|
| 栅格可见 | 显示栅格 |
| 栅格捕捉 | 启用捕捉 |
| 栅格间距 | X/Y方向间距 |
| 自动连接 | 启用自动导线连接 |
| T节点 | 自动T节点连接 |

### 3.3.3 编号设置

| 设置项 | 说明 |
|--------|------|
| 设备标识 | 设备符号前缀定义 |
| 线号格式 | 导线的编号规则 |
| 端子编号 | 端子排编号规则 |

## 3.4 项目操作

### 3.4.1 打开项目

**方法1**：菜单「项目」→「打开」
**方法2**：快捷键 Ctrl+O
**方法3**：双击.elk文件

### 3.4.2 保存项目

**方法1**：菜单「项目」→「保存」
**方法2**：快捷键 Ctrl+S
**方法3**：自动保存（需在设置中启用）

### 3.4.3 备份项目

| 备份方式 | 说明 |
|----------|------|
| 手动备份 | 「项目」→「备份」 |
| 自动备份 | 设置定时自动备份 |
| 压缩备份 | 可压缩项目文件 |

### 3.4.4 项目清理

「项目」→「压缩数据」可清理：

- 未使用的符号
- 未使用的设备
- 查找结果列表
- 消息日志

## 3.5 多项目管理

### 3.5.1 主项目和子项目

```
主项目
├── 子项目1
├── 子项目2
└── 子项目3
```

### 3.5.2 项目复制

「项目」→「复制」可完整复制项目

# 第4章 原理图绘制

## 4.1 页面操作

### 4.1.1 新建页面

**步骤1**：执行「页」→「新建」

**步骤2**：设置页面属性：

| 属性 | 说明 |
|------|------|
| 页名 | 页面名称/标题 |
| 页号 | 页面序号 |
| 页类型 | 功能原理图、安装图等 |
| 图框 | 使用的图框模板 |
| 结构标识 | 页面所属结构 |

### 4.1.2 页面复制

**方法1**：选中页面 → Ctrl+C → Ctrl+V
**方法2**：选中页面 → 右键「复制页」

### 4.1.3 页面排序

在项目导航器中拖拽页面可调整顺序

## 4.2 符号插入

### 4.2.1 从符号库插入

**步骤1**：执行「插入」→「符号」或点击工具栏图标

**步骤2**：在符号选择器中：
- 按类别浏览符号
- 使用搜索功能查找
- 预览符号

**步骤3**：选择符号后在图纸上点击放置

### 4.2.2 符号类别

| 类别 | 包含符号 |
|------|----------|
| 主电路 | 断路器、接触器、热继电器、电机 |
| 控制电路 | 按钮、指示灯、继电器线圈 |
| 供电 | 变压器、电源、熔断器 |
| 测量仪表 | 电流表、电压表、功率表 |
| 保护 | 避雷器、接地 |
| 连接件 | 端子、插头、插座 |
| PLC | PLC输入/输出、电源模块 |
| 网络 | 总线符号 |

### 4.2.3 插入符号示例

**插入断路器**：
1. 执行「插入」→「符号」
2. 类别选择「主电路」→「断路器」
3. 选择合适的断路器符号
4. 在图纸上点击放置
5. 按ESC退出插入模式

### 4.6.3 交叉引用设置
